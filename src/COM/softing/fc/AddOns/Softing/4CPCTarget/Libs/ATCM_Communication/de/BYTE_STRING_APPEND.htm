BYTE_STRING_APPEND

Hängt einen Bytestring an einen anderen Bytestring an.

Schnittstelle

FUNCTION BYTE_STRING_APPEND
    VAR_IN_OUT
        BSTR1 : BYTE_STRING;
    END_VAR
    VAR_INPUT
        BSTR2 : BYTE_STRING;
    END_VAR

Parameter

IN_OUT

BSTR1
Instanz von BYTE_STRING
Bytestring, an den der an BSTR2 übergebene Bytestring angehängt wird.

INPUT

BSTR2
Instanz von BYTE_STRING
Bytestring, der an den an BSTR1 übergebenen Bytestring angehängt wird.

Beschreibung

BYTE_STRING_APPEND hängt den an BSTR2 übergebenen Bytestring an den an BSTR1 übergebenen Bytestring an. Die Größe des an BSTR1 übergebenen Bytestrings wird an die für das Anhängen erforderliche Größe angepaßt.
