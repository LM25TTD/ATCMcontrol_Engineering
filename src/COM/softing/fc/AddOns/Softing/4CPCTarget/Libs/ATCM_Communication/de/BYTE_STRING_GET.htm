BYTE_STRING_GET

Liest den Wert eines Bytes in einem Bytestring.

Schnittstelle

FUNCTION BYTE_STRING_GET : BYTE
VAR_INPUT 
    BSTR  : BYTE_STRING;
    INDEX : DINT;
END_VAR

Rückgabewert

Wert vom Typ BYTE.
Wert des gelesenen Bytes im Bytestring.

Parameter

INPUT

BSTR
Instanz der Struktur BYTE_STRING.
Bytestring, aus dem ein Wert gelesen werden soll.
INDEX
Wert vom Typ DINT
Definiert den Index des Bytes, dessen Inhalt aus dem Bytestring gelesen werden soll.

Beschreibung

BYTE_STRING_GET gibt den Wert des Bytes im Bytestring von BSTR zurück, das sich an der in Index definierten Stelle des Bytestrings befindet.

Vor dem Aufruf von BYTE_STRING_GET muß die als Parameter übergebene Instanz von BYTE_STRING entweder mit der Funktion BYTE_STRING_INIT initialisiert, oder die Strukturvariable size von der an BSTR übergebenen Instanz von BYTE_STRING auf einen Wert >= INDEX gesetzt werden, da beim Erzeugen einer Instanz von BYTE_STRING die Größe des Bytestrings standardmäßig mit dem Wert 0 initialisiert wird. Dies hat zur Folge, daß beim Aufruf von BYTE_STRING_GET mit INDEX > 0 auf einen nicht existierenden Index zugegriffen und daher ein Laufzeitfehler ausgelöst wird.
