BYTE_STRING_SET

Ändert den Wert eines Bytes in einem Bytestring.

Schnittstelle

FUNCTION BYTE_STRING_SET
VAR_IN_OUT 
    BSTR  : BYTE_STRING; 
END_VAR
VAR_INPUT 
    INDEX : DINT;
    VALUE : BYTE;
END_VAR

Parameter

IN_OUT

BSTR
Instanz der Struktur BYTE_STRING.
Bytestring, in dem ein Wert geändert werden soll.

INPUT

INDEX
Wert vom Typ DINT.
Definiert den Index des Bytes, dessen Wert im Bytestring geändert werden soll.
VALUE
Wert vom Typ BYTE.
Definiert den zu schreibenden Wert.

Beschreibung

BYTE_STRING_SET schreibt den Wert von VALUE an das in INDEX angegebene Byte des an BSTR übergebenen Bytestrings.

Vor dem Aufruf von BYTE_STRING_SET muß die als Parameter übergebene Instanz von BYTE_STRING entweder mit der Funktion BYTE_STRING_INIT initialisiert, oder die Strukturvariable size der an BSTR übergebenen Instanz von BYTE_STRING auf einen Wert >= INDEX gesetzt werden, da beim Erzeugen einer Instanz von BYTE_STRING die Größe des Bytestrings standarmäßig mit dem Wert 0 initialisiert wird. Dies hat zur Folge, daß beim Aufruf von BYTE_STRING_GET mit INDEX > 0 auf einen nicht existierenden Index zugegriffen und daher ein Laufzeitfehler ausgelöst wird.
