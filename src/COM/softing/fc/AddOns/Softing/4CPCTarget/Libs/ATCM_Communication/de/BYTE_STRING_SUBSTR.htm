BYTE_STRING_SUBSTR

Extrahiert einen Teilstring aus dem übergebenen Ausgangs-Bytestring.

Schnittstelle

FUNCTION BYTE_STRING_SUBSTR
    VAR_INPUT
        BSTR : BYTE_STRING;
        POS : DINT;
        NUM : DINT;
    END_VAR
    VAR_IN_OUT
        BSTR_SUB : BYTE_STRING;
    END_VAR

Parameter

INPUT

BSTR
Instanz von BYTE_STRING
Bytestring, aus dem ein Teilstring extrahiert werden soll.
POS
Wert vom Typ DINT
Definiert die Position, ab der der Teilstring aus dem an BSTR übergebenen Ausgangstring extrahiert werden soll. Die Position bezeichnet das nte Byte des Ausgangsstrings. Das erste Byte im Ausgangstring hat die Position 0.
NUM
Wert vom Typ DINT
Definiert die Anzahl Bytes, die aus dem an BSTR übergebenen Ausgangsstrings extrahiert werden sollen.

IN_OUT

BSTR_SUB
Instanz von BYTE_STRING
Bytestring, in dem der aus BSTR extrahierte Teilstring gespeichert wird.

Beschreibung

BYTE_STRING_SUB extrahiert aus dem an BSTR übergebenen Ausgangsstring ab der in POS spezifizierten Position einen Teilstring der in NUM spezifizierten Länge und legt den Teilstring in BSTR_SUB ab.

Enthält der Ausgangsstring ab der in POS angegebenen Position bis zum Ende des Ausgangsstrings weniger Bytes, als in NUM spezifiziert wurde, wird nur der Bereich in den Ergebnisstring kopiert, der innerhalb der in POS spezifizierten Position und dem letzten Byte des Ausgangsstrings liegt.

Wird in POS eine Startposition für die Extraktion des Teilstrings spezifiziert, die hinter dem letzten Byte des Ausgangsstrings liegt, wird nichts in den an BSTR_SUB übergebenen Teilstring kopiert. Der Teilstring hat in diesem Fall die Größe 0 Bytes.
