BYTE_STRING_TO_WSTRING

Konvertiert einen Bytestring in einen WSTRING.

Schnittstelle

FUNCTION BYTE_STRING_TO_WSTRING : WSTRING
VAR_INPUT 
    BSTR  : BYTE_STRING;
END_VAR

Rückgabewert

Wert vom Typ WSTRING.
Enthält den in einen WSTRING konvertierten Bytestring.

Parameter

INPUT

BSTR
Instanz der Struktur BYTE_STRING.
Bytestring, der in einen WSTRING konvertiert werden soll.

Beschreibung

BYTE_STRING_TO_WSTRING konvertiert die in der als Parameter übergebenen Instanz von BYTE_STRING abgelegte Bytekette in einen WSTRING. Die Funktion gibt den konvertierten Bytestring zurück.

BYTE_STRING_TO_WSTRING eignet sich nicht zum Lesen von Zeichen, die nicht im ASCII-Zeichensatz enthalten sind, wie z.B. Umlauten. Verwenden Sie diese Funktion daher nur zum Auslesen von Bytestrings bei der seriellen Kommunikation und bei der Kommunikation über UDP. Für das Lesen von Textdateien mit Bausteinen der Bibliothek 4C_FileAccess steht zur Konvertierung von Bytestrings in WSTRINGs die Funktion COPY_WSTRING_FROM_BYTE_STRING zur Verfügung.
