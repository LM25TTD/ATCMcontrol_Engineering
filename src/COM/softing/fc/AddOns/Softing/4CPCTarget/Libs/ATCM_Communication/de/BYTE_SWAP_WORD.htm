BYTE_SWAP_WORD

Wandelt den übergebenen Wert vom Big-Endian- in das Little-Endian-Format und umgekehrt.

Schnittstelle

FUNCTION BYTE_SWAP_WORD : WORD
VAR_INPUT
    VALUE : WORD;
END_VAR

Rückgabewert

Wert vom Typ WORD
Gewandelter Wert von VALUE.

Parameter

INPUT

VALUE
Wert vom Typ WORD
Spezifiziert den zu wandelnden Wert.

Beschreibung

BYTE_SWAP_WORD dreht die Reihenfolge der Bytes innerhalb des an VALUE übergebenen Wertes um und liefert das Ergebnis als Rückgabewert. Zweck der Umkehrungsoperation ist, den übergebenen Wert vom Big-Endian- in das Little-Endian-Format zu wandeln und umgekehrt.
