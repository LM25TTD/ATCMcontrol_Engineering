COMM_CHANNEL

Repräsentiert einen Kommunikationskanal für serielle Kommunikation.

Schnittstelle

TYPE
     COMM_CHANNEL : STRUCT
        hChannel    : DINT;             
        iCommType   : DINT;             
        iTimeout    : DINT;              
     END_STRUCT;
END_TYPE

Parameter

hChannel

Wert vom Typ DINT.
Enthält den Handle des Kanals.

iCommType

Wert vom Typ DINT.
Verwendete Kommunikationsart. Derzeit wird nur serielle Kommunikation unterstützt. Einziger gültiger Wert ist deshalb die Konstante CFB_Serial oder der entsprechende Wert 0.

iTimeout

Wert vom Typ DINT.
Enthält den Timeout für Schreib- und Lesezugriffe in Millisekunden.

Beschreibung

COMM_CHANNEL repräsentiert einen Kommunikationskanal. Eine Instanz von COMM_CHANNEL wird beim Herstellen einer Verbindung mit CONNECT automatisch erzeugt und mit Werten initialisiert. Um den zu verwendenden Kommunikationskanal zu identifizieren, wird bei Read- und Write-Requests mit Instanzen von SER_READ und SER_WRITE die beim Herstellen der Verbindung generierte Instanz von COMM_CHANNEL an deren Parameter ID übergeben.
