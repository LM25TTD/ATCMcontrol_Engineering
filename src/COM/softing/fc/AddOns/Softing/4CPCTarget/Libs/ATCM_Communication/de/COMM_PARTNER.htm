COMM_PARTNER

Repräsentiert eine Verbindung.

Schnittstelle

TYPE
     COMM_PARTNER : STRUCT
        iCommType   : DINT := CFB_Serial;   
        iTimeout    : DINT := 0;            
        sParam      : WSTRING;              
     END_STRUCT;
END_TYPE

Parameter

iCommType

Wert vom Typ DINT.
Definiert die zu verwendende Kommunikationsart. Derzeit wird nur serielle Kommunikation unterstützt. Einziger gültiger Wert ist deshalb die Konstante CFB_Serial oder der entsprechende Wert 0. Dieser Wert ist der Standardinitialwert.

iTimeout

Wert vom Typ DINT.
Definiert den Timeout für Read- und Write-Requests und das Herstellen einer Verbindung in Millisekunden. Bei Initialisierung mit dem Wert 0 wird kein Timeout überwacht.
Standardwert ist 0.
Über den Timeout wird festgelegt, wie lange versucht wird, einen Read- oder Write-Request durchzuführen oder eine Verbindung herzugstellen, bevor der Versuch abgebrochen wird. Der in iTimeout festgelegte Wert gilt für alle Funktionsbausteine, die eine Operation auf eine Instanz von COMM_PARTNER ausführen.

sParam

Wert vom Typ WSTRING.
String, der die Verbindungsparameter spezifiziert. Beim Öffnen einer neuen Verbindung werden mit Hilfe des in sParam übergebenen Verbindungsstrings die Parameter der Schnittstelle definiert. Zu Syntax und möglichen Einstellungen im Verbindungsstring siehe Abschnitt Verbindungsparameter.

Beschreibung

COMM_PARTNER repräsentiert eine Verbindung. Eine Instanz von COMM_PARTNER wird an eine Instanz von CONNECT übergeben, um eine Verbindung herzustellen. Vor der Übergabe an CONNECT müssen die Strukturelemente der entsprechenden Instanz von COMM_PARTNER initialisiert werden.

Zentrale Eigenschaften der Verbindung werden im Verbindungsstring des Elements sParam festgelegt. Die Verbindungseigenschaften sind vom Öffnen bis zum Schließen einer Verbindung gültig. Änderungen der Werte einer Instanz von COMM_PARTNER werden nur beim Öffnen einer Verbindung wirksam. Auf eine bereits geöffnete Verbindung haben Änderungen an Werten der Strukturelemente keine Auswirkung.
