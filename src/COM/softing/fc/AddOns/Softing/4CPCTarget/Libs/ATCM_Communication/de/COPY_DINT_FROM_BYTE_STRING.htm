COPY_DINT_FROM_BYTE_STRING

Liest einen Wert vom Typ DINT an einer bestimmten Position innerhalb eines Bytestrings.

Schnittstelle

FUNCTION COPY_DINT_FROM_BYTE_STRING : DINT
    VAR_IN_OUT
        POS : DINT;
        BSTR : BYTE_STRING;
    END_VAR

Rückgabewert

Wert vom Typ DINT
Wert, der aus dem Bytestring gelesen wurde.

Parameter

IN_OUT

POS
Wert vom Typ DINT
Spezifiziert, ab welcher Stelle im Bytestring der Wert gelesen werden soll. Die Zählung beginnt bei 0 für die erste Postion im Bytestring und erfolgt in der Einheit Bytes.
Nach erfolgreicher Ausführung der Funktion wird POS mit dem Wert der ersten Position im Bytestring, die auf das letzte gelesene Byte folgt, überschrieben. War die Ausführung nicht erfolgreich, hat POS den Wert -1.
BSTR
Instanz der Struktur BYTE_STRING
Bytestring, aus dem ein Wert gelesen werden soll.

Beschreibung

COPY_DINT_FROM_BYTE_STRING liest ab der an POS übergebenen Position einen Wert vom Typ DINT aus dem an BSTR übergebenen Bytestring, und liefert diesen als Rückgabewert zurück.
