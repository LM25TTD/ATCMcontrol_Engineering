COPY_TOD_TO_BYTE_STRING

Schreibt einen Wert vom Typ TIME_OF_DAY an eine bestimmte Stelle innerhalb eines Bytestrings.

Schnittstelle

FUNCTION COPY_TOD_TO_BYTE_STRING
    VAR_INPUT
        VALUE : TIME_OF_DAY;
    END_VAR
    VAR_IN_OUT
        POS : DINT;
        BSTR : BYTE_STRING;
    END_VAR

Parameter

INPUT

VALUE
Wert vom Typ TIME_OF_DAY
Wert, der an der in POS übergebenen Position des Bytestrings eingefügt werden soll.

IN_OUT

POS
Wert vom Typ DINT
Spezifiziert, an welcher Stelle im Bytestring der an VALUE übergebene Wert eingefügt werden soll. Die Zählung beginnt bei 0 für die erste Postion im Bytestring und erfolgt in der Einheit Bytes.
Nach erfolgreicher Ausführung der Funktion wird POS mit dem Wert der ersten auf das letzte geschriebene Byte folgenden Position im Bytestring überschrieben. War die Ausführung nicht erfolgreich, hat POS den Wert -1.
BSTR
Instanz der Struktur BYTE_STRING
Bytestring, in den ein Wert geschrieben werden soll.

Beschreibung

COPY_TOD_TO_BYTE_STRING schreibt an der an POS übergebenen Position einen Wert vom Typ TIME_OF_DAY in den an BSTR übergebenen Bytestring. Werte, die sich im Bytestring an der in POS spezifierten Position befinden, werden mit dem in VALUE angegebenen Wert überschrieben.

Ist der übergebene Bytestring zu klein, um den in VALUE übergebenen Wert aufzunehmen, oder ist die Position, an der der Wert eingefügt werden soll, größer als die aktuelle Größe des Bytestrings, wird der Bytestring auf die minimale erforderliche Größe erweitert.
