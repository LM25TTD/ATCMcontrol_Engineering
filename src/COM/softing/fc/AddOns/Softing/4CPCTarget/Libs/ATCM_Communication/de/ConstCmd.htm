UDP-Befehlskonstanten

Konstanten für die zuletzt über einen Socket ausgeführte Operation.

Konstante Numerischer Wert Beschreibung
UDP_CMD_OPEN 0 Der Socket wurde initialisiert.
UDP_CMD_CLOSE 1 Der Socket wurde geschlossen.
UDP_CMD_SEND 2 Eine Botschaft wurde gesendet.
UDP_CMD_RCV 3 Eine Botschaft wurde empfangen.
UDP_CMD_LOOKUP 4 Es wurde eine IP-Adresse zu einem Hostnamen ermittelt.

Beschreibung

Die UDP-Befehlskonstanten definieren Werte für die Struktur-Variable LastCmd der Struktur UDP_SOCKET_HANDLE.
