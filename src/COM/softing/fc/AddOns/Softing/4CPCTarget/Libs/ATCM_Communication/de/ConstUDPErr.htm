UDP-Fehlerkonstanten

Konstanten zur Auswertung von Fehlern beim Datentransfer mittels UDP.

Konstante Numerischer Wert Beschreibung
UDP_ERR_SOCK_INIT_FAILED -201 Der Socket konnte nicht initialisiert werden.
UDP_ERR_SOCK_NOT_OPENED -202 Der Socket wurde vor dem Versuch, Daten zu senden oder zu empfangen, nicht initialisiert.
UDP_ERR_UNKNOWNHOST -203 Die angegebene Host-Adresse konnte nicht gefunden werden.
UDP_ERR_SEND -204 Ein IO-Fehler ist während des Versuchs, eine Botschaft zu senden, aufgetreten.
UDP_ERR_RECV -205 Ein IO-Fehler ist während des Versuchs, eine Botschaft zu lesen, aufgetreten.
UDP_ERR_RECV_MSG_LONGER -206 Das gelesene Paket enthält mehr Bytes, als im Parameter DATA_LEN angegeben. Die über den in DATA_LEN gesetzten Wert hinausliegenden Bytes werden abgeschnitten.
UDP_ERR_SEND_MSG_TOO_LONG -207 Die Botschaft ist zu lang, um sie über einen UDP-Socket zu transportieren.
UDP_ERR_SEND_HOST_UNREACH -208 Der angegebene Kommunikationspartner kann zur Zeit nicht erreicht werden.
UDP_ERR_LOOKUP_RUNNING -209 Ein an eine Instanz des asynchronen Funktionsbausteins UDP_LOOKUP übergebener Parameter wurde geändert, obwohl die von dieser Instanz ausgeführte Operation noch nicht abgeschlossen ist. In diesem Fehlerfall werden die während der Ausführung der Operation geänderten Parameter ignoriert, und die Operation arbeitet weiterhin mit den ursprünglich übergebenen Parametern.
UDP_ERR_LOOKUP_HN_CHANGE -210 Der an eine Instanz des asynchronen Funktionsbausteins UDP_LOOKUP als Parameter übergebene Hostname wurde nach Abschluß der von dieser Instanz ausgeführten Operation geändert, obwohl keine neue Suche nach einer IP-Adresse mit einer steigenden Flanke am Parameter EN_L gestartet wurde. Dieser Fehlerwert soll darauf hinweisen, daß der vom Ausgang HOST_ADDRESS gelieferte Wert nicht die IP-Adresse des aktuell an den Parameter HOST_NAME übergebenen Rechners ist, sondern die IP-Adresse des Rechners, die bei Abschluß der zuletzt ausgeführten Suchoperation gefunden wurde.
UDP_ERR_PORTNR_OUTOFRANGE -212 Die als Parameter übergebene Portnummer liegt nicht im gültigen Bereich von 0 bis 65535.

Beschreibung

Die UDP-Fehlerkonstanten definieren Werte für das Element LastErrorNr der Struktur UDP_SOCKET_HANDLE und den Parameter ERROR_NR der UDP-Funktionsbausteine.
