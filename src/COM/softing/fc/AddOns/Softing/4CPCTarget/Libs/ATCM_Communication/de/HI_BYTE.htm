HI_BYTE

Extrahiert aus dem höherwertigen Anteil eines Werts vom Datentyp WORD einen neuen Wert vom Datentyp BYTE.

Schnittstelle

FUNCTION HI_BYTE : BYTE
    VAR_INPUT
        VALUE : WORD;
    END_VAR

Rückgabewert

Wert vom Typ BYTE
Aus dem höherwertigen Anteil von VALUE neu generierter Wert.

Parameter

INPUT

VALUE
Wert vom Typ WORD
Wert, aus dem der höherwertige Anteil extrahiert werden soll.

Beschreibung

HI_BYTE extrahiert aus dem höherwertigen Anteil des an VALUE übergebenen Werts einen neuen Wert vom Datentyp BYTE und liefert das Ergebnis als Rückgabewert zurück.
