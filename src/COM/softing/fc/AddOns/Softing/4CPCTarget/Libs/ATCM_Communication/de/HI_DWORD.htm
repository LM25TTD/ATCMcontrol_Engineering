HI_DWORD

Extrahiert aus dem höherwertigen Anteil eines Werts vom Datentyp LWORD einen neuen Wert vom Datentyp DWORD.

Schnittstelle

FUNCTION HI_DWORD : DWORD
    VAR_INPUT
        VALUE : LWORD;
    END_VAR

Rückgabewert

Wert vom Typ DWORD
Aus dem höherwertigen Anteil von VALUE neu generierter Wert.

Parameter

INPUT

VALUE
Wert vom Typ LWORD
Wert, aus dem der höherwertige Anteil extrahiert werden soll.

Beschreibung

HI_DWORD extrahiert aus dem höherwertigen Anteil des an VALUE übergebenen Werts einen neuen Wert vom Datentyp DWORD und liefert das Ergebnis als Rückgabewert zurück.
