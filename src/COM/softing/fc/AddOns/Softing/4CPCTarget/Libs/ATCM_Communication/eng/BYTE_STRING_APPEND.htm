BYTE_STRING_APPEND

Appends a bytestring to another bytestring.

Interface definition

FUNCTION BYTE_STRING_APPEND
    VAR_IN_OUT
        BSTR1 : BYTE_STRING;
    END_VAR
    VAR_INPUT
        BSTR2 : BYTE_STRING;
    END_VAR

Parameters

IN_OUT

BSTR1
Instance of BYTE_STRING
Bytestring that is append to the bytestring passed in BSTR2.

INPUT

BSTR2
Instance of BYTE_STRING
Bytestring to which the bytestring passed in BSTR1 is appended.

Description

BYTE_STRING_APPEND appends the bytestring passed in BSTR2 to the bytestring passed in BSTR1 . The size of the bytestring passed in BSTR1 is adjusted to the size needed for appending the bytestring.
