BYTE_STRING_SUBSTR

Extracts a substring from the passed output bytestring.

Interface definition

FUNCTION BYTE_STRING_SUBSTR
    VAR_INPUT
        BSTR : BYTE_STRING;
        POS : DINT;
        NUM : DINT;
    END_VAR
    VAR_IN_OUT
        BSTR_SUB : BYTE_STRING;
    END_VAR

Parameters

INPUT

BSTR
Instance of BYTE_STRING
Bytestring, from which a substring is to be extracted.
POS
Value of type DINT
Defines the position from which on the substring is to be extracted. The position marks the nth byte of the output string. The first byte in this string has the position 0.
NUM
Value of type DINT
Defines the number of bytes, that should be extracted out if the output string passed to BSTR.

IN_OUT

BSTR_SUB
Instance of BYTE_STRING
Bytestring, to which the substring extracted from BSTR is saved to.

Description

BYTE_STRING_SUB extracts a substring of a length specified in NUM from a position in the output string passed to BSTR specified in POS and saves the substring in BSTR_SUB.

If the output string contains less bytes from the position defined in POS on than specified in NUM, only the array that lies between the position specified in POS and and the last byte of the output string is copied into the result string.

If a starting position is specified in POS for extraction that lies beyond the last byte of the output string, nothing is copied into the substring passed to BSTR_SUB. The substring has the size 0 bytes in this case.
