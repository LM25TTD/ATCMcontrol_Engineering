COMM_CHANNEL

Represents a communication channel for serial communication.

Interface definition

TYPE
     COMM_CHANNEL : STRUCT
        hChannel    : DINT;             
        iCommType   : DINT;             
        iTimeout    : DINT;              
     END_STRUCT;
END_TYPE

Parameters

hChannel

Value of type DINT.
Determines the handle of the channel used.

iCommType

Value of type DINT.
Determines the type of communication. At present, only serial communication is supported.

iTimeout

Value of type DINT.
Determines the timeout for read and write requests. If iTimeout has the value 0, no timeout is used.

Description

COMM_CHANNEL represents a communication channel. An instance of COMM_CHANNEL is generated and initialized with values automatically when a connection is established with CONNECT. In order to indentify the communication channel used, the instance of COMM_CHANEL, which has been generated when the connection was established, is passed to the paramter ID of SER_READ and SER_WRITE in case of a read or write request.
