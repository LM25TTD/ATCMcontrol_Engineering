COMM_PARTNER

Represents a connection.

Interface definition

TYPE
     COMM_PARTNER : STRUCT
        iCommType   : DINT := CFB_Serial;   
        iTimeout    : DINT := 0;            
        sParam      : WSTRING;              
     END_STRUCT;
END_TYPE

Parameters

iCommType

Value of type DINT.
Determines the type of communication. At present, only CFB_SERIAL is supported.

iTimeout

Value of type DINT.
Determines the timeout for read and write requests.
The default value is 0, i.e. no timeout is used.

sParam

Value of type WSTRING.
String containing parameters for initializing the connection. When a new connection is opened, the parameters of the interface are defined using the connection string passed in sParam. For information regarding syntax and possible settings click here.

Description

COMM_PARTNER represents a connection. An instance of COMM_PARTNER is passed to an instance of CONNECT in order to establish a connection. Before it can be passed to CONNECT, the structure elements of the corresponding instance of COMM_PARTNER have to be initialized.

Basic properties of the connection are defined in the connection string of the element sParam. The connection properties are valid from the beginning till the end of a connection. A change of the values of an instance of COMM_PARTNER only takes place when a connection is being opened. Changes to the values of the structure elements have no effects on an already established connection.
