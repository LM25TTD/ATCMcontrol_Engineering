UDP-Command constants

Constants for the last operation executed using a socket.

Constant
Numeric value
 Description
UDP_CMD_OPEN 0 The socket was initialized.
UDP_CMD_CLOSE 1 The socket was closed.
UDP_CMD_SEND 2 A message was sent.
UDP_CMD_RCV 3 A message was received
UDP_CMD_LOOKUP 4 An IP-adress of a host name was detected.

Description

The UDP-Comand constants define values for the structure variable LastCmd of the structure UDP_SOCKET_HANDLE.
