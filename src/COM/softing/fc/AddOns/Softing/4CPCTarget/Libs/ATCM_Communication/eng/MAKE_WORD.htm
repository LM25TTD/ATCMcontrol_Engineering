MAKE_WORD

Generates a new unsigned integer value of type WORD from two values of type BYTE.

Interface definition

FUNCTION MAKE_WORD : WORD
    VAR_INPUT
        LO_BYTE : BYTE;
        HI_BYTE : BYTE;
    END_VAR

Return value

Value of type WORD
Value generated from LO_BYTE and HI_BYTE.

Parameters

INPUT

LO_BYTE
Value of type BYTE
Lower significant proportion of the value to be generated.
HI_BYTE
Value of type BYTE
Higher significant proportion of the value to be generated.

Description

MAKE_WORD combines the integer values passed to LO_BYTE and HI_BYTE to a new unsigned integer value with the amount of bytes doubled and returns the result. The value in LO_BYTE defines the lower siginificant and the value in HI_BYTE the higher significant proportion of the new value to be generated.
