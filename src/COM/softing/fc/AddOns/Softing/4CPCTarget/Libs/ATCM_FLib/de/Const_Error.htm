Fehlerkonstanten

Konstanten für die Fehlerauswertung im Parameter ERRCD aller asynchronen Bausteine und im Parameter LSTERRCD der Strukturen FA_ASYNC_FILE und FA_SYNC_FILE.

Konstante Numerischer Wert Beschreibung
  0 Kein Fehler aufgetreten.
CFA_CommonError 10 Allgemeiner Fehler. Weitere Informationen zu dem aufgetretenen Fehler können über den Parameter ERRTXT der Bibliotheks-Bausteine und über den Parameter LSTERRTXT der Strukturen FA_ASYNC_FILE und FA_SYNC_FILE ausgewertet werden.
CFA_InternalError 11 Interner Fehler der unterliegenden Java-Implementation. Dieser Fehler kann nur bei der Arbeit mit den elementaren sychronen Bausteinen auftreten.
CFA_InvalidArgument 20 Dem aufgerufenen Baustein wurde ein ungültiger Parameter übergeben.
CFA_FileNotOpenError 30 Die Datei, auf die eine Operation ausgeführt werden soll, ist nicht geöffnet. Die Operation kann nur auf eine bereits geöffnete Datei ausgeführt werden.
CFA_Timeout 40 Der für die Dateioperation festgelegte Timeout wurde überschritten.
CFA_FileAlreadyOpen 50 Mit der Strukturinstanz ist bereits eine Datei geöffnet worden. Mit einer Strukturinstanz kann nur eine Datei geöffnet sein.
