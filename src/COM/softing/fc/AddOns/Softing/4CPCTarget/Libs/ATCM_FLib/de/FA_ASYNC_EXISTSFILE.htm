FA_ASYNC_EXISTSFILE

Prüft, ob eine Datei existiert.

Schnittstelle

FUNCTION_BLOCK FA_ASYNC_EXISTSFILE
VAR_INPUT
    EXEC        : BOOL;             
END_VAR
VAR_IN_OUT
    FILE        : FA_ASYNC_FILE;    
END_VAR
VAR_OUTPUT
    EXISTS      : BOOL;             
    DONE        : BOOL;             
    ERROR       : BOOL;
    ERRCD       : DINT;
    ERRTXT      : WSTRING;
END_VAR

Parameter

INPUT

EXEC
Wert vom Typ BOOL.
Startet die Operation bei steigender Flanke.

IN_OUT

FILE
Instanz der Struktur FA_ASYNC_FILE.
Dateiobjekt, auf das die Operation ausgeführt wird.

OUTPUT

EXISTS
Wert vom Typ BOOL
Zeigt nach Ausführung der Operation an, ob die im Element FILENAME der an FILE übergebenen Strukturinstanz von FA_ASYNC_FILE spezifizierte Datei existiert.
Mögliche Werte:
TRUE: Die Datei existiert.
FALSE: Die Datei existiert nicht.
DONE
Wert vom Typ BOOL.
Signalisiert den (erfolgreichen oder gescheiterten) Abschluß der Operation.
Mögliche Werte:
TRUE: Operation ist abgeschlossen.
FALSE: Operation ist noch nicht abgeschlossen.
ERROR
Wert vom Typ BOOL
Signalisiert den Fehlerstatus.
Mögliche Werte:
TRUE: Ein Fehler ist aufgetreten.
FALSE: Es ist kein Fehler aufgetreten.
ERRCD
Wert vom Typ DINT
Enthält eine Fehlernummer, falls ein Fehler aufgetreten ist.
Ist kein Fehler aufgetreten, hat ERRCD den Wert 0.
Mögliche Werte sind in den Fehlerkonstanten definiert.
ERRTXT
Wert vom Typ WSTRING
Enthält einen Fehlertext, falls ein Fehler aufgetreten ist.
Ist kein Fehler aufgetreten, enthält ERRTXT einen Leerstring.

Beschreibung

FA_ASYNC_EXISTSFILE überprüft, ob eine Datei existiert.

Der Name der zu suchenden Datei wird FA_ASYNC_EXISTSFILE über den Parameter FILE übergeben. FILE erwartet eine Instanz der Struktur FA_ASYNC_FILE. Diese Struktur enthält das Strukturelement FILENAME, das einen Verzeichnis- oder Dateinamen spezifiziert und vor Übergabe der Strukturinstanz an FA_ASYNC_EXISTSFILE initialisiert werden muß.

Der Aufruf von FA_ASYNC_EXISTSFILE ist unabhängig von einem Zugriffszyklus, d.h. es muß kein vorhergehender Aufruf von FA_ASYNC_OPENFILE erfolgen.
