FA_ASYNC_GETSIZE

Liefert die Größe einer Datei in Bytes.

Schnittstelle

FUNCTION_BLOCK FA_ASYNC_GETSIZE
VAR_INPUT
    EXEC        : BOOL;             
END_VAR
VAR_IN_OUT
    FILE        : FA_ASYNC_FILE;    
END_VAR
VAR_OUTPUT
    SIZE        : LINT;             
    DONE        : BOOL;             
    ERROR       : BOOL;
    ERRCD       : DINT;
    ERRTXT      : WSTRING;
END_VAR

Parameter

INPUT

EXEC
Wert vom Typ BOOL
Startet die Operation bei steigender Flanke.

IN_OUT

FILE
Instanz der Struktur FA_ASYNC_FILE.
Dateiobjekt, auf das die Operation ausgeführt wird.

OUTPUT

SIZE
Wert vom Typ LINT
Enthält nach Beendigung der Operation die Größe der in FILE übergebenen Datei in Bytes.
DONE
Wert vom Typ BOOL
Signalisiert den (erfolgreichen oder gescheiterten) Abschluß der Operation.
Mögliche Werte:
TRUE: Operation ist abgeschlossen.
FALSE: Operation ist noch nicht abgeschlossen.
ERROR
Wert vom Typ BOOL
Signalisiert den Fehlerstatus.
Mögliche Werte:
TRUE: Ein Fehler ist aufgetreten.
FALSE: Es ist kein Fehler aufgetreten.
ERRCD
Wert vom Typ DINT
Enthält eine Fehlernummer, falls ein Fehler aufgetreten ist.
Ist kein Fehler aufgetreten, hat ERRCD den Wert 0.
Mögliche Werte sind in den Fehlerkonstanten definiert.
ERRTXT
Wert vom Typ WSTRING
Enthält einen Fehlertext, falls ein Fehler aufgetreten ist.
Ist kein Fehler aufgetreten, enthält ERRTXT einen Leerstring.

Beschreibung

FA_ASYNC_GETSIZE liefert im Parameter SIZE die Größe der an FILE übergebenen Datei. Die Datei muß vor dem Ermitteln der Dateigröße nicht geöffnet werden, d.h. die Ausführung von FA_ASYNC_GETSIZE ist unabhängig von einem Zugriffszyklus.
