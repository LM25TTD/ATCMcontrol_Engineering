FA_SYNC_CLOSEFILE

Schließt eine geöffnete Datei.

Schnittstelle

FUNCTION FA_SYNC_CLOSEFILE:BOOL
VAR_INPUT
    EXEC        : BOOL;             
END_VAR
VAR_IN_OUT
    FILE        : FA_SYNC_FILE;     
END_VAR

Rückgabewert

Wert vom Typ BOOL
Signalisiert den Fehlerstatus.
Mögliche Werte:
TRUE: Ein Fehler ist aufgetreten.
FALSE: Es ist kein Fehler aufgetreten.

Parameter

INPUT

EXEC
Wert vom Typ BOOL
Startet die Operation.

IN_OUT

FILE
Instanz der Struktur FA_SYNC_FILE
Dateiobjekt, auf das die Operation ausgeführt wird.

Beschreibung

FA_SYNC_CLOSEFILE schließt die an FILE übergebene Datei. FA_SYNC_CLOSEFILE muß immer aufgerufen werden, um einen Zugriffszyklus abzuschließen. Die zu schließende Datei wird durch eine Instanz der Struktur FA_SYNC_FILE repräsentiert.
