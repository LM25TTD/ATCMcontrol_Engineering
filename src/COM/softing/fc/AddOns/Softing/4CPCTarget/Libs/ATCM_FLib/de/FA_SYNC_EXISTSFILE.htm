FA_SYNC_EXISTSFILE

Prüft, ob eine Datei existiert.

Schnittstelle

FUNCTION FA_SYNC_EXISTSFILE:BOOL
VAR_INPUT
    EXEC        : BOOL;             
END_VAR
VAR_IN_OUT
    FILE        : FA_SYNC_FILE;     
END_VAR
VAR_OUTPUT
    EXISTS      : BOOL;             
END_VAR

Rückgabewert

Wert vom Typ BOOL
Signalisiert den Fehlerstatus.
Mögliche Werte:
TRUE: Ein Fehler ist aufgetreten.
FALSE: Es ist kein Fehler aufgetreten.

Parameter

INPUT

EXEC
Wert vom Typ BOOL
Startet die Operation.

IN_OUT

FILE
Instanz der Struktur FA_SYNC_FILE
Im Gegensatz zu den meisten anderen Funktionen der Bausteingruppe für elementare synchrone Dateioperationen repräsentiert die an FILE übergebene Strukturinstanz nicht ein Dateiobjekt, auf das die betreffenden Dateioperation ausgeführt wird. Stattdessen wird das Element FILENAME der Struktur FA_SYNC_FILE für die Definition des Dateinamens der zu suchenden Datei genutzt.

OUTPUT

EXISTS
Wert vom Typ BOOL
Zeigt nach Ausführung der Operation an, ob die im Element FILENAME der an FILE übergebenen Strukturinstanz von FA_SYNC_FILE spezifizierte Datei existiert.
Mögliche Werte:
TRUE: Die Datei existiert.
FALSE: Die Datei existiert nicht.

Beschreibung

FA_SYNC_EXISTSFILE überprüft, ob eine Datei existiert.

Der Name der zu suchenden Datei wird FA_SYNC_EXISTSFILE über den Parameter FILE übergeben. FILE erwartet eine Instanz der Struktur FA_SYNC_FILE. Diese Struktur enthält das Strukturelement FILENAME, das einen Dateinamen spezifiziert und vor Übergabe der Strukturinstanz an FA_SYNC_EXISTSFILE initialisiert werden muß.

Der Aufruf von FA_SYNC_EXISTSFILE ist unabhängig von einem Zugriffszyklus, d.h. es muß kein vorhergehender Aufruf von FA_SYNC_OPENFILE erfolgen.
