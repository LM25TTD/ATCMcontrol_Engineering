FA_SYNC_GETSIZE

Liefert die Größe einer Datei in Bytes.

Schnittstelle

FUNCTION FA_SYNC_GETSIZE:BOOL
VAR_INPUT
    EXEC        : BOOL;            
END_VAR
VAR_IN_OUT
    FILE        : FA_SYNC_FILE;     
END_VAR
VAR_OUTPUT
    SIZE        : LINT;                 
END_VAR

Rückgabewert

Wert vom Typ BOOL
Signalisiert den Fehlerstatus.
Mögliche Werte:
TRUE: Ein Fehler ist aufgetreten.
FALSE: Es ist kein Fehler aufgetreten.

Parameter

INPUT

EXEC
Wert vom Typ BOOL
Startet die Operation.

IN_OUT

FILE
Instanz der Struktur FA_SYNC_FILE
Dateiobjekt, auf das die Operation ausgeführt wird.

OUTPUT

SIZE
Wert vom Typ LINT.
Enthält nach Beendigung der Operation die Größe der in FILE übergebenen Datei in Bytes.

Beschreibung

FA_SYNC_GETSIZE liefert im Parameter SIZE die Größe der an FILE übergebenen Datei. Die Datei muß vor dem Ermitteln der Dateigröße nicht geöffnet werden, d.h. die Ausführung von FA_SYNC_GETSIZE ist unabhängig von einem Zugriffszyklus.
