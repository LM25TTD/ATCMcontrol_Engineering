FA_SYNC_OPENFILE

Öffnet eine Datei zur Bearbeitung.

Schnittstelle

FUNCTION FA_SYNC_OPENFILE:BOOL
VAR_INPUT
    EXEC        : BOOL;             
END_VAR
VAR_IN_OUT
    FILE        : FA_SYNC_FILE;     
END_VAR

Rückgabewert

Wert vom Typ BOOL
Signalisiert den Fehlerstatus.
Mögliche Werte:
TRUE: Ein Fehler ist aufgetreten.
FALSE: Es ist kein Fehler aufgetreten.

Parameter

INPUT

EXEC
Wert vom Typ BOOL
Startet die Operation.

IN_OUT

FILE
Instanz der Struktur FA_SYNC_FILE.
Dateiobjekt, auf das die Operation ausgeführt wird.

Beschreibung

FA_SYNC_OPENFILE öffnet die an FILE übergebene Datei für Lese- oder Schreibzugriff. FA_SYNC_OPENFILE leitet einen Zugriffzyklus ein und muß daher immer aufgerufen werden, bevor Lese- oder Schreibzugriffe über andere Funktionsbausteininstanzen der Bausteingruppe für elementare synchrone Dateioperationen erfolgen können.

Die zu öffnende Datei wird durch eine Instanz der Struktur FA_SYNC_FILE repräsentiert. Diese Struktur enthält das Element MODE, über das festgelegt wird, mit welchem Zugriffsmodus eine Datei geöffnet wird. Mögliche Werte sind in den Zugriffsmodus-Konstanten definiert.

Vor dem Öffnen einer Datei muß der Zugriffsmodus in der an FA_SYNC_OPENFILE übergebenen Strukturinstanz von FA_SYNC_FILE gesetzt werden. Der beim Öffnen gesetzte Zugriffsmodus ist solange gültig, bis die Datei mit FA_SYNC_CLOSEFILE wieder geschlossen wird. Alle zwischen dem Öffnen und Schließen einer Datei aufgerufenen Funktionen, die auf das an FA_SYNC_OPENFILE übergebene Dateiobjekt arbeiten, unterliegen den beim Öffnen der Datei in MODE gesetzten Zugriffsberechtigungen. Wertzuweisungen an MODE, die innerhalb eines mit FA_SYNC_OPENFILE eingeleiteten Zugriffszyklus vorgenommen werden, haben für die Dauer des gesamten Zugriffszyklus keine Wirkung. Es ist daher nicht möglich, während eines Zugriffszyklus den Zugriffsmodus zu verändern.

Die Position des Dateizeigers ist beim Öffnen einer Datei immer abhängig von dem in der übergebenen Strukturinstanz von FA_SYNC_FILE gesetzten Zugriffsmodus. Eine Initialisierung der Elementvariablen POSITION der Struktur FA_SYNC_FILE mit einem benutzerdefinierten Wert wird beim Öffnen mit FA_SYNC_OPENFILE durch den dem entsprechenden Zugriffsmodus zugeordneten Wert überschrieben. Sollen Dateizugriffe nach dem Öffnen einer Datei an einer bestimmten Position erfolgen, muß daher nach dem Öffnen der Wert von POSITION explizit neu gesetzt werden.
