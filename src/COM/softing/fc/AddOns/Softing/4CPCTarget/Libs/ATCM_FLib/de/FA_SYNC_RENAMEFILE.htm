FA_SYNC_RENAMEFILE

Benennt eine Datei um.

Schnittstelle

FUNCTION FA_SYNC_RENAMEFILE:BOOL
VAR_INPUT
    EXEC        : BOOL;             
    NEWNAME     : WSTRING;          
END_VAR
VAR_IN_OUT
    FILE        : FA_SYNC_FILE;     
END_VAR

Rückgabewert

Wert vom Typ BOOL
Signalisiert den Fehlerstatus.
Mögliche Werte:
TRUE: Ein Fehler ist aufgetreten.
FALSE: Es ist kein Fehler aufgetreten.

Parameter

INPUT

EXEC
Wert vom Typ BOOL
Startet die Operation.
NEWNAME
Wert vom Typ WSTRING
Dateiname, in den die Datei umbenannt werden soll.
Der Dateiname kann mit voll qualifiziertem Pfad angegeben werden. Wird kein Pfad spezifiziert, wird intern das Standard-Arbeitsverzeichnis von ATCMControl als Pfad ergänzt.
Wenn ein anderer Pfad als der im Parameter FILENAME der an FILE übergebenen Strukturinstanz von FA_SYNC_FILE spezifizierte Pfad angegeben wird, wird die umbenannte Datei in das in NEWNAME angegebene Verzeichnis verschoben.
Existiert die in NEWNAME spezifizierte Datei bereits, wird die Operation abgebrochen und ein Fehler gesetzt.
Der Dateiname einschließlich Pfadangabe wird vom Compiler nicht auf korrekte Syntax oder Plausibilität überprüft.

IN_OUT

FILE
Instanz der Struktur FA_SYNC_FILE
Dateiobjekt, auf das die Operation ausgeführt wird.

Beschreibung

FA_SYNC_RENAMEFILE benennt eine Datei in den im Parameter NEWNAME spezifizierten Dateinamen um. Die Datei, die umbenannt werden soll, wird im Parameter FILENAME der an FA_SYNC_RENAMEFILE übergebenen Instanz von FA_SYNC_FILE angegeben. Nachdem die Datei umbenannt wurde, enthält dieser Parameter den neuen Dateinamen.

FA_SYNC_RENAMEFILE bildet unter der Bausteingruppe für elementare synchrone Dateioperationen eine Ausnahme. Im Gegensatz zu den meisten anderen Funktionen dieser Gruppe darf eine Instanz von FA_SYNC_RENAMEFILE nicht innerhalb eines Zugriffzyklus aufgerufen werden, da nur das Umbenennen von Dateien erlaubt ist, auf die aktuell kein Zugriff durch einen anderen Prozeß erfolgt. Die an FA_SYNC_RENAMEFILE übergebene Datei darf deshalb nicht über FA_SYNC_OPENFILE oder von einem Fremdprogramm aktuell geöffnet sein.
