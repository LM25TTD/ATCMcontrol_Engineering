FA_SYNC_WRITEFILE

Schreibt einen Bytestring in eine Datei.

Schnittstelle

FUNCTION FA_SYNC_WRITEFILE:BOOL     
VAR_INPUT
    EXEC        : BOOL;             
    DATA        : BYTE_STRING;      
END_VAR
VAR_IN_OUT
    FILE        : FA_SYNC_FILE;     
END_VAR

Rückgabewert

Wert vom Typ BOOL
Signalisiert den Fehlerstatus.
Mögliche Werte:
TRUE: Ein Fehler ist aufgetreten.
FALSE: Es ist kein Fehler aufgetreten.

Parameter

INPUT

EXEC
Wert vom Typ BOOL
Startet die Operation.
DATA
Instanz der Struktur BYTE_STRING
Spezifiziert den zu schreibenden Wert.
Verwenden Sie die Funktion COPY_WSTRING_TO_BYTE_STRING, um den Bytestring mit einem Wert zu initialisieren.
Zur Konvertierung eine Bytestrings in einen WSTRING können Sie die Funktion COPY_WSTRING_FROM_BYTE_STRING aus der Bibliothek 4C_Communication verwenden.

IN_OUT

FILE
Instanz der Struktur FA_SYNC_FILE
Dateiobjekt, auf das die Operation ausgeführt wird.

Beschreibung

FA_SYNC_WRITEFILE schreibt den an DATA übergebenen Wert in die an FILE übergebene Datei. Vor dem Schreiben in eine Datei muß diese mit FA_SYNC_OPENFILE geöffnet werden. Für den Schreibzugriff muß vor dem Öffnen der Datei der Zugriffmodus im Parameter MODE der an FA_SYNC_OPENFILE übergebenen Strukturinstanz von FA_SYNC_FILE auf CFA_READ_WRITE oder CFA_APPEND gesetzt werden.

Der Dateizeiger, der die aktuelle Schreibposition markiert, wird durch die Strukturvariable POSITION der an FILE übergebenen Strukturinstanz von FA_SYNC_FILE repräsentiert. Der erste auf das Öffnen einer Datei folgende Schreibzugriff erfolgt immer an der Position, die durch den beim Öffnen der Datei gesetzten Zugriffsmodus definiert ist, wenn der Dateizeiger nicht explizit auf eine andere Position gesetzt wird (siehe unten).

Nach jedem Schreibzugriff wird der Wert von POSITION automatisch auf eine Position direkt hinter dem zuletzt geschriebenen Byte gesetzt. Auf diese Weise wird gewährleistet, daß in eine Datei sequentiell geschrieben werden kann, ohne daß der Datezeiger explizit auf die neue Position gesetzt werden muß.

Die Position eines Schreibzugriffs kann durch Zuweisung eines Werts an die Strukturvariable POSITION explizit gesetzt werden. Dazu muß nach dem Öffnen einer Datei und vor dem Schreibzugriff der Wert von POSITION auf die gewünschte Schreibposition gesetzt werden. Nach Ende der Schreiboperation wird der Wert von POSITION wieder auf die sich direkt hinter dem zuletzt geschriebenen Byte befindende Position gesetzt.
