Error Constants

Constants for error evaluation in the parameter ERRCD of all asynchronous blocks and in the parameter LSTERRCD of the FA_ASYNC_FILE and FA_SYNC_FILE structures.

Constant Numeric value Description
  0 No error has occurred.
CFA_CommonError 10 General error. Further information on the error that has occurred can be evaluated via the ERRTXT parameter of the library blocks and via the LSTERRTXT parameter of the FA_ASYNC_FILE and FA_SYNC_FILE structures.
CFA_InternalError 11 Internal error of the underlying Java implementation. This error can only occur when you are working with the elementary sychronous blocks.
CFA_InvalidArgument 20 An invalid parameter was passed to the called block.
CFA_FileNotOpenError 30 The file on which an operation is to be performed has not been opened. The operation can only be performed on a file that has already been opened.
CFA_Timeout 40 The timeout defined for the file operation was exceeded.
CFA_FileAlreadyOpen 50 A file has already been opened with the structure instance. Only one file may be opened with a structure instance.
