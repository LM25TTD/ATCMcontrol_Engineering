Position Constants

Constants for the reading and writing position within a file.

Constant Numeric value Description
CFA_BOF 0 Beginning of file. First valid reading or writing position.
CFA_EOF -1 End of file
CFA_EOL -2 End of line

Description

Position constants define values for the structure variable POSITION of the FA_ASYNC_FILE and FA_SYNC_FILE structures and for the IN_OUT variable POSITION of the function blocks of the group of complex asynchronous blocks.
