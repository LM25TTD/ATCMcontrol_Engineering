FA_ASYNC_EXISTSFILE

Checks whether a file exists.

Interface

FUNCTION_BLOCK FA_ASYNC_EXISTSFILE
VAR_INPUT
    EXEC        : BOOL;             
END_VAR
VAR_IN_OUT
    FILE        : FA_ASYNC_FILE;    
END_VAR
VAR_OUTPUT
    EXISTS      : BOOL;             
    DONE        : BOOL;             
    ERROR       : BOOL;             
    ERRCD       : DINT;                  
    ERRTXT      : WSTRING;
END_VAR

Parameter

INPUT

EXEC
Value of the type BOOL.
Starts the operation at a rising edge.

IN_OUT

FILE
Instance of the FA_ASYNC_FILE structure.
File object on which the operation is performed.

OUTPUT

EXISTS
Value of the type BOOL
Indicates, after executing the operation, whether the file specified in the element FILENAME of the structure instance of FA_ASYNC_FILE passed to FILE exists.
Possible values:
TRUE: The file exists.
FALSE: The file does not exist.
DONE
Value of the type BOOL.
Signalizes the (successful or failed) conclusion of the operation.
Possible values:
TRUE: Operation has been concluded.
FALSE: Operation has not yet been concluded.
ERROR
Value of the type BOOL
Signalizes the error status.
Possible values:
TRUE: An error has occurred.
FALSE: No error has occurred.
ERRCD
Value of the type DINT
Contains an error number if an error has occurred.
If no error has occurred, ERRCD has the value 0.
The possible values are defined in the error constants.
ERRTXT
Value of the type WSTRING
Contains an error text if an error has occurred.
If no error has occurred, ERRTXT contains an empty string.

Description

FA_ASYNC_EXISTSFILE checks whether a file exists.

The name of the file to be found is passed to FA_ASYNC_EXISTSFILE in the parameter FILE. FILE expects an instance of the FA_ASYNC_FILE structure. This structure includes the structure element FILENAME which specifies a directory or file name and must be initialized before the structure instance is passed to FA_ASYNC_EXISTSFILE .

The call of FA_ASYNC_EXISTSFILE is independent of an access cycle, which means that a previous call of FA_ASYNC_OPENFILE is not required.
