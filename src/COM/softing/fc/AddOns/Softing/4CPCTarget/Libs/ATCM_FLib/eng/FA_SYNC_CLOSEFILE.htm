FA_SYNC_CLOSEFILE

Closes an open file.

Interface

FUNCTION FA_SYNC_CLOSEFILE:BOOL
VAR_INPUT
    EXEC        : BOOL;
END_VAR
VAR_IN_OUT
    FILE        : FA_SYNC_FILE;
END_VAR

Return value

Value of the type BOOL
Signalizes the error status.
Possible values:
TRUE: An error has occurred.
FALSE: No error has occurred.

Parameter

INPUT

EXEC
Value of the type BOOL
Starts the operation.

IN_OUT

FILE
Instance of the FA_SYNC_FILE structure
File object on which the operation is performed.

Description

FA_SYNC_CLOSEFILE closes the file passed to FILE. FA_SYNC_CLOSEFILE must always be called to terminate an access cycle. The file to be closed is represented by an instance of the FA_SYNC_FILE structure.
