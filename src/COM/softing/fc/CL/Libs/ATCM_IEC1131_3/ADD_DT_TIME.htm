ADD_DT_TIME

Adds a value of type TIME to a value of type DT.

Interface definition

FUNCTION ADD_DT_TIME : DT
VAR_INPUT  
    IN1  : DT;  
    IN2  : TIME; 
END_VAR

Return value

Value of type DT
Result of the addition of IN1 and IN2.

Parameters

INPUT

IN1
Value of type DT
Value to be added.
IN2
Value of type TIME
Value to be added.

Description

Yields the result of adding IN1 and IN2. Instead of using the ADD_DT_TIME function, you can also use the plus operator + to add a value of type DATE_AND_TIME and a value of type TIME.
