ADD_TIME

Adds two values of the type TIME.

Interface definition

FUNCTION ADD_TIME : TIME
VAR_INPUT  
    IN1 : TIME;  
    IN2 : TIME; 
END_VAR

Return value

Value of type TIME
Result of the addition of IN1 and IN2.

Parameters

INPUT

IN1
Value of type TIME
Value to be added.
IN2
Value of type TIME
Value to be added.

Description

Yields the result of adding IN1 and IN2. Instead of using the ADD_TIME function, you can also use the plus operator + or the ADD function to add values of type TIME.
