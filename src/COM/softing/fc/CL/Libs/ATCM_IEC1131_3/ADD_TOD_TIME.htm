ADD_TOD_TIME

Adds a value of type TIME to a value of type TOD.

Interface definition

FUNCTION ADD_TOD_TIME : TOD
VAR_INPUT  
    IN1 : TOD;  
    IN2 : TIME; 
END_VAR

Return value

Value of type TOD
Result of the addition of IN1 und IN2.

Parameters

INPUT

IN1
Value of type TOD
Value to be added.
IN2
Value of type TIME
Value to be added.

Description

Yields the result of adding IN1 and IN2. Instead of using the ADD_TOD_TIME function, you can also use the plus operator + to add a value of type TIME_OF_DAY and a value of type TIME.
