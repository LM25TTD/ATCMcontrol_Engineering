AND

Combines the values delivered as actual parameters with bitwise AND.

Interface definition

FUNCTION AND : ANY_BIT
VAR_INPUT 
    IN1 : ANY_BIT; 
END_VAR
VAR_IN_EXT 
    IN2 : ANY_BIT; 
END_VAR

Return value

Value of type ANY_BIT
Yields the value resulting from combining the actual parameters.

Parameters

INPUT

IN1
Value of type ANY_BIT
Value to be combined.
IN2 (IN_EXT)
Value of type ANY_BIT
Value to be combined.
IN_EXT-variables are extensible. Optional, further values can be stated to be combined.

Description

The function AND corresponds to the operator for bitwise AND. With a bitwise AND, the bit patterns of the output value and the bit mask are combined in such a way that, as the result of the operation, a bit is set only at that position at which a bit had been set in both values, the output value and bit mask alike. The bitwise AND can therefore be used for precisely disabling specific bits or for testing whether specific bits have been set.
