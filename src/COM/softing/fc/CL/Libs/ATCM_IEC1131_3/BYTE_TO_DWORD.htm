BYTE_TO_DWORD

Converts the value of a variable of type BYTE to the data type DWORD.

Interface definition

FUNCTION BYTE_TO_DWORD : DWORD
VAR_INPUT
    IN : BYTE;
END_VAR

Return value

Value of type DWORD
Result of the conversion.

Parameters

INPUT

IN
Value of type BYTE
Value to be converted.

Description

The function BYTE_TO_DWORD converts a value of type BYTE to a value of type DWORD.
