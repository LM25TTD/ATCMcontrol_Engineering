CONCAT

Joins the strings passed as parameters together to make up one complete string.

Interface definition

FUNCTION CONCAT : ANY_STRING
VAR_INPUT  
    IN1 : ANY_STRING; 
END_VAR
VAR_IN_EXT 
    IN2 : ANY_STRING; 
END_VAR

Return value

Value of type ANY_STRING
Complete string from the joined strings.

Parameters

INPUT

IN1
Value of type ANY_STRING
String to which another string is to be added.
IN2 (IN_EXT)
Value of type ANY_STRING
String to be added.
IN_EXT-variables are extensible, i.e. in addition to IN2, further strings can be passed for joining.

Description

Joins the strings passed as parameters together to make up one complete string.
