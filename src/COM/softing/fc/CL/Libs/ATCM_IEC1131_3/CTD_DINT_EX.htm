CTD_DINT_EX

Down counter for rising edges with additional visualization functionality.

Interface definition

FUNCTION_BLOCK CTD_DINT_EX
VAR_INPUT  
    CD         : BOOL R_EDGE; 
    LD         : BOOL;        
    PV         : DINT;        
END_VAR

VAR_INPUT  
    SCALEMIN   : DINT:=0;     
    SCALEMAX   : DINT:=100;   
END_VAR
VAR_OUTPUT 
    Q          : BOOL;        
    CV         : DINT;
    ERROR	: DINT;           
END_VAR

Parameters

INPUT

CD
Value of type BOOL with the attribute R_EDGE
Input signal that is evaluated for edge detection.
LD
Value of type BOOL
Resets the edge counter CV to the value of PV if TRUE.
PV
Value of type DINT
 Value to which the counter variable CV is set if LD is TRUE.
SCALEMIN
Value of type DINT
Determines the lower value for the visualization scale. The default value is 0.
SCALEMAX
Value of type DINT
Determines the upper value for the visualization scale. The default value is 100.

OUTPUT

Q
Value of type BOOL
Flag that is set to TRUE when the counter variable CV reaches a value <=0.
CV
Value of type DINT
 Counts rising edges at input CD.
ERROR
Value of type DINT
Yields the value 0 if no error occurred. An error number <> 0 is returned if an error occurred during visualization. In the current version of 4CONTROL, only the event that the current value of SCALEMIN >= SCALEMAX is encountered is evaluated as a visualization error.
 

Description

This function block is used as a down counter for rising edges at the input signal CD. The current count of down counting is stored in the counter variable CV. The counter variable is reset to the value passed in PV when the flag LD is set to the value TRUE. This flag is not reset automatically. If counting is to be restarted after the reset, the flag must be reset to FALSE beforehand. The variable Q is set to TRUE when CV reaches a value <=0.

CTD_DINT_EX is the extended version of the function block CTD_DINT. As additional parameters, this function block features the two inputs SCALEMIN and SCALEMAX, which can be used for setting the scale of values of a bar chart represented in the faceplate, and the output ERROR, which returns an error code if a visualization error occurred.
