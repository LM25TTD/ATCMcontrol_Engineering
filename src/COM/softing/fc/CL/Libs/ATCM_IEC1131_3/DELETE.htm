DELETE

Deletes a set number of characters in a string at a set position.

Interface definition

FUNCTION DELETE : ANY_STRING
VAR_INPUT  
    IN : ANY_STRING; 
    L  : ANY_INT;
    P  : ANY_INT;
END_VAR

Return value

Value of type ANY_STRING
Edited string

Parameters

INPUT

IN
Value of type ANY_STRING
String in which characters are to be deleted.
L
Value of type ANY_INT
Number of characters to be deleted.
P
Value of type ANY_INT
Position of the character in IN1 as from which characters are to be deleted.

Description

Deletes the number of characters specified in L in the passed string, starting from position P. If L or P contain a value that exceeds the number of characters contained in the string, a runtime error is generated and the calling task is stopped.
