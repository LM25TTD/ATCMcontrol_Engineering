DIV

Divides one value by an other value.

Interface definition

FUNCTION DIV : ANY_NUM
VAR_INPUT 
    IN1: ANY_NUM; 
    IN2: ANY_NUM; 
END_VAR

Return value

Value of Type ANY_NUM
Result of the division.

Parameters

INPUT

IN1
Value of type ANY_NUM
Value to be divided by IN2.
IN2
Value of type ANY_NUM
Value by which IN1 is divided.

Description

Divides IN1 by IN2.
Corresponds to the operator /.
