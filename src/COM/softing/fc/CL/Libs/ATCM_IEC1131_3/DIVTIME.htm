DIVTIME

Divides a value of type TIME.

Interface definition

FUNCTION DIVTIME : TIME
VAR_INPUT  
    IN1 : TIME;  
    IN2 : ANY_NUM; 
END_VAR

Return value

Value of type TIME
Result of the division of IN1 by IN2 in ms.

Parameters

INPUT

IN1
Value of type TIME
Value which is to be divided by IN2.
IN2
Value of type ANY_NUM
Value by which IN1 is divided.

Description

Divides IN1 by IN2 and yields the result in ms. Instead of the DIV_TIME function, you can also use the division operator / to perform a division on a value of data type TIME.
