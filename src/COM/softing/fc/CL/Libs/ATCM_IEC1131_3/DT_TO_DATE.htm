DT_TO_DATE

Converts the value of a variable of type DT to the data type DATE.

Interface definition

FUNCTION DT_TO_DATE : DATE
VAR_INPUT
    IN : DT;
END_VAR

Return value

Value of type DATE
Result of the conversion.

Parameters

INPUT

IN
Value of type DT
Value to be converted.

Description

The function DT_TO_DATE converts a value of type DT to a value of type DATE.
