EQ

Compares two values and returns TRUE if both are equal.

Interface definition

FUNCTION EQ : BOOL
VAR_INPUT 
    IN1 : ANY_ELEMENTARY; 
    IN2 : ANY_ELEMENTARY; 
END_VAR

Return value

Value of type BOOL
TRUE if IN1 = IN2.

Parameters

INPUT

IN1
Value of type ANY_ELEMENTARY
Value to be compared.
IN2
Value of type ANY_ELEMENTARY
Value to be compared.

Description

Returns TRUE if both values are equal; else, FALSE is returned. Corresponds to the operator =.
