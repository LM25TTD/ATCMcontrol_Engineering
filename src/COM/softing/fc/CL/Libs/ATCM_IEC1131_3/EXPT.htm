EXPT

Raises IN1 to the power of IN2.

Interface definition

FUNCTION EXPT : ANY_REAL
VAR_INPUT 
    IN1: ANY_REAL; 
    IN2: ANY_NUM; 
END_VAR

Return value

Value of type ANY_REAL
Exponent of the value IN1.

Parameters

INPUT

IN1
