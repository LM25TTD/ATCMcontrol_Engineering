FIND

Searches a string for a set character string.

Interface definition

FUNCTION FIND_DINT : ANY_INT
VAR_INPUT  
    IN1 : ANY_STRING; 
    IN2 : ANY_STRING; 
END_VAR

Return value

Value of type ANY_INT
Yields the position of the sought string.

Parameters

INPUT

IN1
Value of type ANY_STRING
String in which the search is to be performed.
IN2
Value of type ANY_STRING
Character string which is to be found.
 

Description

Returns the position in IN1 at which the character string of IN2 is found for the first time.
If IN2 is not found, the value 0 is returned.
