GE

Compares two values and returns TRUE if the first value is greater than/equal to the second value.

Interface definition

FUNCTION GE : BOOL
VAR_INPUT 
    IN1 : ANY_ELEMENTARY; 
    IN2 : ANY_ELEMENTARY; 
END_VAR

Return value

Value of type BOOL
TRUE if IN1 >= IN2.

Parameters

INPUT

IN1
Value of type ANY_ELEMENTARY
Value to be compared.
IN2
Value of type ANY_ELEMENTARY
Value to be compared.

Description

Returns TRUE if the first value is greater than/equal to the second value; else, FALSE is returned. Corresponds to the operator >=.
