INSERT

Inserts a string into another string at a set position.

Interface definition

FUNCTION INSERT_DINT : ANY_STRING
VAR_INPUT  
    IN1 : ANY_STRING; 
    IN2 : ANY_STRING; 
    P   : ANY_INT;
END_VAR

Return value

Value of type ANY_STRING
Complete string from the joined strings.

Parameters

INPUT

IN1
Value of type ANY_STRING
String in which the string passed in IN2 is inserted.
IN2
Value of type ANY_STRING
String which is inserted in the string passed in IN1.
P
Value of type ANY_INT
Position of the character in IN1 after which IN2 is to be inserted in IN1.

Description

Inserts the string passed in IN2 after position P of IN1. If P contains a value that exceeds the number of characters contained in IN1, a runtime error is generated and the calling task is stopped.
