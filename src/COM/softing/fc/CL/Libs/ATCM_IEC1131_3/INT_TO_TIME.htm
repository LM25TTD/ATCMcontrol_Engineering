INT_TO_TIME

Converts the value of a variable of type INT to the data type TIME.

Interface definition

FUNCTION INT_TO_TIME : TIME
VAR_INPUT
    IN : INT;
END_VAR

Return value

Value of type TIME
Result of the conversion.

Parameters

INPUT

IN
Value of type INT
Value to be converted.

Description

The function INT_TO_TIME converts a value of type INT to a value of type TIME.
