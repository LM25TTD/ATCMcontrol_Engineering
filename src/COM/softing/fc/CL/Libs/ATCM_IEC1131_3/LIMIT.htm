LIMIT

Used for limiting a value.

Interface definition

FUNCTION LIMIT : ANY_ELEMENTARY
VAR_INPUT 
    MN : ANY_ELEMENTARY; 
    IN : ANY_ELEMENTARY; 
    MX : ANY_ELEMENTARY; 
END_VAR

Return value

Value of type ANY_ELEMENTARY
A value out of the range determined by MN and MX.

Parameters

INPUT

MN
Value of type ANY_ELEMENTARY
Limits the lower value range.
IN
Value of type ANY_ELEMENTARY
Returned if the value is not outside the value range specified in MN and MX.
MX
Value of type ANY_ELEMENTARY
Limits the upper value range.

Description

The limit values are specified in the first (MN) and third (MX) parameters. As the second parameter (IN), the value that is passed is to be checked as to whether it exceeds or falls below the limit values. If this value is outside the specified limits, the value defined as the upper or lower limit is returned, depending on whether the value range was exceeded or fallen below. If the value range is fallen below, the value of the lower limit is returned; if the value exceeds the value range, the value of the upper limit is returned. If the value is within the value range defined in the first and third parameters, the value is returned unchanged.
