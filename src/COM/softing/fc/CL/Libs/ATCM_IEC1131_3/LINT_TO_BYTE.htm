LINT_TO_BYTE

Converts the value of a variable of type LINT to the data type BYTE.

Interface definition

FUNCTION LINT_TO_BYTE : BYTE
VAR_INPUT
    IN : LINT;
END_VAR

Return value

Value of type BYTE
Result of the conversion.

Parameters

INPUT

IN
Value of type LINT
Value to be converted.

Description

The function LINT_TO_BYTE converts a value of type LINT to a value of type BYTE.
