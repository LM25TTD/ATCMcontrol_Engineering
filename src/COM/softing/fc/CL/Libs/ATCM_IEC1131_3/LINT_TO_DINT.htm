LINT_TO_DINT

Converts the value of a variable of type LINT to the data type DINT.

Interface definition

VAR_INPUT
    IN : LINT;
END_VAR

Return value

Value of type DINT
Result of the conversion.

Parameters

INPUT

IN

Description

The function LINT_TO_DINT converts a value of type LINT to a value of type DINT.
