LOG

Yields the result of the logarithm to the base 10.

Interface definition

FUNCTION LOG : ANY_REAL
VAR_INPUT 
    IN : ANY_REAL; 
END_VAR

Return value

Value of type ANY_REAL
Decadic logarithm of IN.

Parameters

INPUT

IN
Value of type ANY_REAL
Value, whose decadic logarithm is to be determined.

Description

Yields the result of the logarithm to the base 10. If the value of the input variable is not within the value range of 0.0 .. 1.79769313486231570E+308, the result of the output value will be -/+ infinity.
