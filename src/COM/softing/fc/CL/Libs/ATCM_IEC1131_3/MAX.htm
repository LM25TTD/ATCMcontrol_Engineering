MAX

Selects the highest of the values passed as parameters.

Interface definition

FUNCTION MAX : ANY_ELEMENTARY
VAR_INPUT 
    IN1 : ANY_ELEMENTARY; 
END_VAR
VAR_IN_EXT
    IN2 : ANY_ELEMENTARY;
END_VAR

Return value

Value of type ANY_ELEMENTARY
The highest among the passed values.

Parameters

INPUT

IN1
Value of type ANY_ELEMENTARY
Value to be compared.
IN2
Value of type ANY_ELEMENTARY
Value to be compared.
IN_EXT-variables are extensible. Optional, further values can be stated to be compared.

Description

The function MAX compares all indicated parameters and issues the largest parameter.
