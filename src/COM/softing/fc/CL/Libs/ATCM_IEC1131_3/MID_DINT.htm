MID_DINT

Returns a set number of characters from a string, starting from a set position in the string.

Interface definition

FUNCTION MID_DINT : ANY_STRING
VAR_INPUT  
    IN : ANY_STRING; 
    L  : DINT;
    P  : DINT;
END_VAR

Return value

Value of type ANY_STRING
Characters from the string of IN

Parameters

INPUT

IN
Value of type ANY_STRING
 String from which the number of characters specified in L is returned.
L
Value of type DINT
Number of characters to be returned.
P
Value of type DINT
Position of the character as from which the string is to be returned.

Description

Returns the number of characters specified in L, starting from the position within IN that is passed in P. If L or P contain a value that exceeds the number of characters contained in the string, a runtime error is generated and the calling task is stopped.
