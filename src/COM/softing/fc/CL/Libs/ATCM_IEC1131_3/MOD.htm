MOD

Yields the residual value of the division of IN1 by IN2.

Interface definition

FUNCTION MOD : ANY_INT
VAR_INPUT 
    IN1: ANY_INT; 
    IN2: ANY_INT; 
END_VAR

Return value

Value of type ANY_INT
Residual value of the division.

Parameters

INPUT

IN1
Value of type ANY_INT
Value to be divided by IN2.
IN2
Value of type ANY_INT
Value by which IN1 is divided.

Description

Yields the remainder of the division of IN1 by IN2.
Corresponds to the operator MOD.
