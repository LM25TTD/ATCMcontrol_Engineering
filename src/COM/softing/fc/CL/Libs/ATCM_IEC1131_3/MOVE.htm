MOVE

Corresponds to an assignment. Returns the value at the input In.

Interface

FUNCTION MOVE: ANY
VAR_INPUT
    In: ANY; 
END_VAR

Return value

Value of the type ANY
Result of the assignment.

Parameter

INPUT

IN
Value of the type ANY
Value that is assigned to the return value.

Description

The MOVE function corresponds to an assignment with the operator :=. The value of the input IN is returned as the result of the function.

This function can be used in the Ladder Diagram programming language to reproduce an IF statement in Structured Text.
