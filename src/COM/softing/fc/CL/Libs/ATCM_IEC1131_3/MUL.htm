MUL

Yields the result of a multiplication of the values passed as parameters.

Interface definition

FUNCTION MUL : ANY_NUM
VAR_INPUT  
    IN1 : ANY_NUM; 
END_VAR
VAR_IN_EXT 
    IN2 : ANY_NUM; 
END_VAR

Return value

Value of type ANY_NUM.
Result of the multiplication.

Parameters

INPUT

IN1
Value of type ANY_NUM
Value to be multiplicated.
IN2
Value of type ANY_NUM
Value to be multiplicated.
IN_EXT-Variables are extensible, i.e. in addition to IN2, further values can be passed for multiplication.

Description

Yields the result of a multiplication of the values passed as parameters.
Corresponds to the operator *.
