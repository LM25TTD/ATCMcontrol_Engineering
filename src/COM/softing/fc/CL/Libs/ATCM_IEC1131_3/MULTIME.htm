MULTIME

Multiplies a value of type TIME.

Interface definition

FUNCTION MULTIME : TIME
VAR_INPUT  
    IN1 : TIME;  
    IN2 : ANY_NUM; 
END_VAR

Return value

Value of type TIME
Result of the multiplication of IN1 with IN2 in ms.

Parameters

INPUT

IN1
Value of type TIME
Value which is multiplicated with IN2.
IN2
Value of type ANY_NUM
Multiplication factor

Description

Multiplies IN1 with IN2 and yields the result in ms. Instead of the MUL_TIME function, you can also use the multiplication operator * to perform a multiplication with a value of type TIME.
