MUX

Returns the value at the index defined in K.

Interface definition

FUNCTION MUX : ANY
VAR_INPUT 
   K : ANY_INT;
   IN0: ANY; 
END_VAR
VAR_IN_EXT
   IN1: ANY; 
END_VAR

Return value

Value of type ANY
The value located at the indicated postion in the index.

Parameters

INPUT

K
Value of type ANY_INT
Index parameter for selection.
IN0
Value of type ANY
First parameter in the index row.
IN1
Value of type ANY
Second parameter in the index row.
IN_EXT-variables are extensible. Optional, further values can be stated to be compared.

Description

The selection of the value is determined by the first parameter (K). This parameter serves as the index designating the position at which the value to be selected is located. The index count starts at 0. Counting begins from the first parameter following the index parameter. If during program run the index parameter is assigned a value higher than the number of parameters to select from, the calling task is stopped. The value returned by the function is set to 0.
