NOT

Sets every bit of the value passed as the actual parameter to its complementary value.

Interface definition

FUNCTION NOT : ANY_BIT
VAR_INPUT
    IN : ANY_BIT;
END_VAR

Return value

Value of type ANY_BIT
Yields the complement of the value passed as a parameter.

Parameters

INPUT

IN
Value of type ANY_BIT
Value to be inverted.

Description

The function NOT corresponds to the operator for the bitwise NOT. All bits of the passed value are set to their complementary value, i.e. if a bit of passed value has the value 1, the bit at the same bit position of the result value receives the value 0, and vice versa. The function NOT accepts only one parameter; it is thus not extensible, in contrast to all other bitwise boolean standard functions.
