REPLACE

Replaces a certain number of characters in a string by characters from another String, beginning from a set position.

Interface definition

FUNCTION REPLACE_DINT : ANY_STRING
VAR_INPUT  
    IN1 : ANY_STRING; 
    IN2 : ANY_STRING; 
    L   : ANY_INT;
    P   : ANY_INT;
END_VAR

Return value

Value of type ANY_STRING
Edited String

Parameters

INPUT

IN1
Value of type ANY_STRING
String in which characters are to be replaced.
IN2
Value of type ANY_STRING
String which contains the characters that are to be inserted in IN1 in place of the characters to be replaced.
L
Value of type ANY_INT
Number of characters to be replaced.
P
Value of type ANY_INT
Position of the character in IN1 as from which characters are to be replaced.

Description

Replaces in IN1 the number of characters specified in L by the characters contained in IN2, starting from position P. If L contains a value that exceeds the number of characters contained in the passed string, starting from the position P, a runtime error is generated and the calling task is stopped. The same applies to the case that P contains a value that exceeds the number of characters contained in the passed string.
