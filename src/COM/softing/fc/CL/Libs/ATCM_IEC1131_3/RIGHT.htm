RIGHT

Returns a set number of characters from a string, starting from the end of the string.

Interface definition

FUNCTION RIGHT : ANY_STRING
VAR_INPUT  
    IN : ANY_STRING; 
    L  : ANY_INT;
END_VAR

Return value

Value of type ANY_STRING
Characters from the string of IN

Parameters

INPUT

IN
Value of type ANY_STRING
 String from which the number of characters specified in L is returned.
L
Value of type ANY_INT
Number of characters to be returned.

Description

Returns the number of characters specified in L, starting from the end of the string (rightmost character) defined in IN. If L contains a value that exceeds the number of characters contained in the string, a runtime error is generated and the calling task is stopped.
