ROL

Rotation of IN by N bit to the left.

Interface definition

FUNCTION ROL : ANY_BIT
VAR_INPUT 
    IN : ANY_BIT; 
    N  : ANY_INT; 
END_VAR

Return value

Value of type ANY_BIT
Result of the Rotation

Parameters

INPUT

IN
Value of type ANY_BIT
 The bit pattern of this value is rotated to the left.
N
Value of type ANY_INT
Specifies the number of bits by which the pattern of IN is to be rotated left.
 

Description

The function ROL rotates the bit pattern of the value passed as a parameter to the left by the specified number of places. This means that all bits are shifted to the left by the specified number of places; bits that are located at the left boundary of the data type's value range are re-inserted at the right boundary of the data type.
