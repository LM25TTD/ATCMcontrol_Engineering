RS

preferrential reset of Q.

Interface definition

FUNCTION_BLOCK RS
VAR_INPUT
   S  : BOOL;
   R1 : BOOL;
END_VAR
VAR_OUTPUT
   Q1 : BOOL;
END_VAR

Parameters

INPUT

S
Value of type BOOL
 Sets Q to TRUE, if R1 = FALSE.
R1
Value of type BOOL
Dominant variable for resetting Q to FALSE.

OUTPUT

Q1
Value of type BOOL
Saves the state resulting from R1 and S.
 

Description
