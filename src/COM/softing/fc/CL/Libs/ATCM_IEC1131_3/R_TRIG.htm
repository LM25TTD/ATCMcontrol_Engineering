R_TRIG

Detects rising edges.

Interface definition

FUNCTION_BLOCK R_TRIG
VAR_INPUT
	CLK: BOOL;
END_VAR
VAR_OUTPUT
	Q: BOOL;
END_VAR

Parameters

INPUT

CLK
Value of type BOOL
 Input signal for edge detection.

OUTPUT

Q
Value of type BOOL
Detects rising edges. Is TRUE for one execution of the function block if the input signal CLK changes from FALSE to TRUE.
 

Description

The function block R_TRIG detects rising edges. Every time the input variable CLK changes from FALSE to TRUE, the output Q is set to TRUE for the execution of a function block call. The next time the function block is called, Q is reset to FALSE again.
