SEL

Selects one of two parameters based on a third parameter.

Interface definition

FUNCTION SEL : ANY
VAR_INPUT 
    G   : BOOL; 
    IN0 : ANY; 
    IN1 : ANY; 
END_VAR

Return value

Value of type ANY
Either IN0 or IN1.

Parameters

INPUT

G
Value of type BOOL
Selects IN1 if TRUE and IN0 if FALSE.
IN0
Value of type ANY
Returned if G = FALSE.
IN1
Value of type ANY
Returned if G = TRUE

Description

Used for choosing between the values passed as the second and third parameters. Which of the two values is selected depends on whether the first parameter yields TRUE or FALSE. If the first parameter is TRUE, the second value (IN1) is returned whereas, if the first parameter is FALSE, the first value (IN0) is returned.
