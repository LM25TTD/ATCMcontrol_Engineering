SHL

Leftshift of IN by N Bit.

Interface definition

FUNCTION SHL : ANY_BIT
VAR_INPUT 
    IN : ANY_BIT; 
    N  : ANY_INT; 
END_VAR

Return value

Value of type ANY_BIT
Result of the shift operation

Parameters

INPUT

IN
Value of type ANY_BIT
 The bit pattern of this value is shifted to the left.
N
Value of type ANY_INT
Specifies the number of bits by which the pattern of IN is to be shifted left.
 

Description

The function SHL shifts the bit pattern of the value passed as a parameter to the left by the specified number of places. On the right, the bits are filled with the value 0 according to the number of shifted places. Bits that exceed the left boundary of the value range specified for the data type, are lost in a left shift.
