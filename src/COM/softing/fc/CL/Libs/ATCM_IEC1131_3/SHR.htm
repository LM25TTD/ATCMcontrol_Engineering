SHR

Rightshift of IN by N Bit.

Interface definition

FUNCTION SHR : ANY_BIT
VAR_INPUT 
    IN : ANY_BIT; 
    N  : ANY_INT; 
END_VAR

Return value

Value of type ANY_BIT
Result of the shift operation.

Parameters

INPUT

IN
Value of type ANY_BIT
 The bit pattern of this value is shifted to the right.
N
Value of type ANY_INT
Specifies the number of bits by which the pattern of IN is to be shifted right.
 

Description

The function SHR shifts the bit pattern of the value passed as a parameter to the right according to the specified number of places. On the left, the bits are filled with the value 0 by the number of shifted places. Bits that exceed the right boundary of the value range specified for the data type, are lost in a right shift.
