SQRT

Yields the square root of the passed parameter.

Interface definition

FUNCTION SQRT : ANY_REAL
VAR_INPUT 
    IN : ANY_REAL; 
END_VAR

Return value

Value of type ANY_REAL
Square root of IN.

Parameters

INPUT

IN
Value of type ANY_REAL
Value, whose square root is to be determined.

Description

Yields the square root of the passed parameter. If the value of the input variable is not within the value range of 0.0 .. 1.79769313486231570E+308, the result of the output value will be -/+ infinity.
