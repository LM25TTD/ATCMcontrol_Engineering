SR

preferrential set of Q.

Interface definition

FUNCTION_BLOCK SR
VAR_INPUT
   S1 : BOOL;
   R  : BOOL;
END_VAR
VAR_OUTPUT
   Q1 : BOOL;
END_VAR

Parameters

INPUT

S1
Value of type BOOL
 Dominant input signal. If TRUE, Q1 is also set to TRUE.
R
Value of type BOOL
Resets Q1 to FALSE if R = TRUE and S1 = FALSE.

OUTPUT

Q1
Value of type BOOL
Saves the state resulting from S1 and R.
 

Description

The function block SR is used for the preferential setting of Q to TRUE if S1 = TRUE. Its over-all function can be taken from the following table:
