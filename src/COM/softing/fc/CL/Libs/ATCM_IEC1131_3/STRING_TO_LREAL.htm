STRING_TO_LREAL

Converts the value of a variable of type STRING to the data type LREAL.

Interface definition

FUNCTION STRING_TO_LREAL : LREAL
VAR_INPUT
    IN : STRING;
END_VAR

Return value

Value of type LREAL
Result of the conversion.

Parameters

INPUT

IN
Value of type STRING
Value to be converted.

Description

The function STRING_TO_LREAL converts a value of type STRING to a value of type LREAL. While converting values of type STRING, errors may occur. More detailed information on this subject may be found here.
