SUB

Yields the result of a subtraction of the values passed as parameters.

Interface definition

FUNCTION SUB : ANY_MAGNITUDE
VAR_INPUT 
    IN1: ANY_MAGNITUDE; 
    IN2: ANY_MAGNITUDE; 
END_VAR

Return value

Value of type ANY_MAGNITUDE
Result of the subtraction.

Parameters

INPUT

IN1
Value of type ANY_MAGNITUDE
Value from which IN2 is subtracted.
IN2
Value of type ANY_MAGNITUDE
Value to be subtracted from IN1.

Description

Subtracts IN2 from IN1. Corresponds to the Operator -.
