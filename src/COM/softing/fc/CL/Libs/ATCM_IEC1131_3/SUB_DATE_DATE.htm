SUB_DATE_DATE

Subtracts two values of type DATE.

Interface definition

FUNCTION SUB_DATE_DATE : TIME
VAR_INPUT  
    IN1 : DATE;  
    IN2 : DATE; 
END_VAR

Return value

Value of type TIME
Result of the subtraction of IN2 from IN1 in ms.

Parameters

INPUT

IN1
Value of type DATE
Value from which IN2 is subtracted.
IN2
Value of type DATE
Value which is subtracted from IN1.

Description

Subtracts IN2 from IN1 and yields the result in ms. Instead of using the SUB_DATE_DATE function, you can also use the minus operator - to subtract one value of type DATE from another.
