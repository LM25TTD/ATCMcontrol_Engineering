SUB_DT_DT

Subtracts two values of type DATE_AND_TIME.

Interface definition

FUNCTION SUB_DT_DT : TIME
VAR_INPUT  
    IN1 : DT;
    IN2 : DT;
END_VAR

Return value

Value of type TIME
Result of the subtraction of IN2 from IN1 in ms.

Parameters

INPUT

IN1
Value of type DT
Value from which IN2 is to be subtracted.
IN2
Value of type DT
Value which is subtracted from IN1.

Description

Subtracts IN2 from IN1 and yields the result in ms. Instead of using the SUB_DT_DT function, you can also use the minus operator - to subtract one value of type DATE_AND_TIME from another.
