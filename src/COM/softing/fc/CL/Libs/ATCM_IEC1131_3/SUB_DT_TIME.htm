SUB_DT_TIME

Subtracts a value of type TIME from a value of type DATE_AND_TIME.

Interface definition

FUNCTION SUB_DT_TIME : DT
VAR_INPUT  
    IN1 : DT;  
    IN2 : TIME; 
END_VAR

Return value

Value of type DT (DATE_AND_TIME)
Result of the subtraction of IN2 from IN1.

Parameters

INPUT

IN1
Value of type DT
Value from which IN2 is to be subtracted.
IN2
Value of type TIME
Value which is to be subtracted from IN1.

Description

Subtracts IN2 from IN1. Instead of using the SUB_DT_TIME function, you can also use the minus operator - to subtract a value of type TIME from a value of type TIME_OF_DAY.
