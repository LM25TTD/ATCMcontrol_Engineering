SUB_TOD_TIME

Subtracts a value of type TIME from a value of type TOD.

Interface definition

FUNCTION SUB_TOD_TIME : TOD
VAR_INPUT  
    IN1 : TOD;  
    IN2 : TIME; 
END_VAR

Return value

Value of type TOD
Result of the subtraction of IN2 from IN1 as value of type TIME_OF_DAY.

Parameters

INPUT

IN1
Value of type TOD
Value from which IN2 is to be subtracted.
IN2
Value of type TIME
Value which is to be subtracted from IN1.

Description

Subtracts IN2 from IN1 and yields the result as time of day. Instead of using the SUB_TOD_TIME function, you can also use the minus operator - to subtract one value of type DATE from another.
