SUB_TOD_TOD

Subtracts two values of type TIME_OF_DAY.

Interface definition

FUNCTION SUB_TOD_TOD : TIME
VAR_INPUT  
    IN1 : TOD;  
    IN2 : TOD; 
END_VAR

Return value

Value of type TIME
Result of the subtraction of IN2 from IN1 in ms.

Parameters

INPUT

IN1
Value of type TOD
Value from which IN2 is to be subtracted.
IN2
Value of type TOD
Value which is to be subtracted from IN1.

Description

Subtracts IN2 from IN1 and yields the result in ms. Instead of using the SUB_TOD_TOD function, you can also use the minus operator - to subtract one value of type TIME_OF_DAY from another.
