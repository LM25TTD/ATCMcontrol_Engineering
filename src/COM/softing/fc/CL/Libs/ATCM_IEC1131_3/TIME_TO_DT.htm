TIME_TO_DT

Converts the value of a variable of type TIME to the data type DT.

Interface definition

VAR_INPUT
    IN : TIME;
END_VAR

Return value

Value of type DT
Result of the conversion.

Parameters

INPUT

IN
Value of type TIME

Description
