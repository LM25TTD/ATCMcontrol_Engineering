TOF_EX

Delayed reaction to an input signal with additional visualization functionality.

Interface definition

FUNCTION_BLOCK TOF_EX

VAR_INPUT   
    IN : BOOL := FALSE; 
    PT : TIME := T#0s;  
END_VAR

VAR_INPUT 
    SCALEMIN : DINT := 0;   
    SCALEMAX : DINT := 100; 
END_VAR

VAR_OUTPUT  
    Q  : BOOL := TRUE; 
    ET : TIME := T#0s; 
    ERROR : DINT;      
END_VAR

Parameters

INPUT

IN
Value of type BOOL
Input signal
The default value is FALSE.
PT
Value of type TIME
Determines the duration of the delay after which Q is set to FALSE if IN = FALSE.
The default value is T#0s.
SCALEMIN
Value of type DINT
Determines the lower value for the visualization scale. The default value is 0.
SCALEMAX
Value of type DINT
Determines the upper value for the visualization scale. The default value is 100.

OUTPUT

Q
Value of type BOOL
Signalizes by the value FALSE that the time specified in PT has expired after IN changed to FALSE. Q is FALSE if ET >= PT.
ET
Value of type TIME
 Is incremented time-dependent after counting has started, provided IN = FALSE ; incrementing is carried out until ET >= PT.
ERROR
Value of type DINT.
Yields the value 0 if no error occurred. An error number <> 0 is returned if an error occurred during visualization. In the current version of 4CONTROL, only the event that the current value of SCALEMIN >= SCALEMAX is encountered is evaluated as a visualization error.

Description

The function block TOF_EX is used for delaying the response to the input signal IN. The length of the delay is determined in PT. If IN changes to FALSE, Q is also set to FALSE as soon as the length of time specified in PT expires.
After reaching the target value in the counting process (ET >= PT), Q will remain FALSE until ET is reset to 0. This is achieved by setting the input IN to TRUE.

TOF_EX is the extended version of the function block TOF. As additional parameters, this function block features the two inputs SCALEMIN and SCALEMAX, which can be used for setting the scale of values of a bar chart represented in the faceplate, and the output ERROR, which returns an error code if a visualization error occurred.
