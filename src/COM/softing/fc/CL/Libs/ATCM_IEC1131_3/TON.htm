TON

Delayed reaction to an input signal.

Interface definition

FUNCTION_BLOCK TON

VAR_INPUT   
    IN : BOOL := FALSE; 
    PT : TIME := T#0s;  
END_VAR

VAR_OUTPUT  
    Q  : BOOL := FALSE; 
    ET : TIME := T#0s;  
END_VAR

Parameters

INPUT

IN
Value of type BOOL
Input signal
The default value is FALSE.
PT
Value of type TIME
Determines the duration of the delay after which Q is set to TRUE if IN = TRUE.
The default value is T#0s.

OUTPUT

Q
Value of type BOOL
Signalizes by the value TRUE that the time specified in PT has expired after IN changed to TRUE. Q is TRUE if ET >= PT.
ET
Value of type TIME
 Is incremented time-dependent after counting has started, provided IN = TRUE ; incrementing is carried out until ET >= PT.

Description

The function block TON is used for delaying the response to the input signal IN. The length of the delay is determined in PT. If IN changes to TRUE, Q is also set to TRUE as soon as the length of time specified in PT expires.
After reaching the target value in the counting process (ET >= PT), Q will remain TRUE until ET is reset to 0. This is achieved by setting the input IN to FALSE.
