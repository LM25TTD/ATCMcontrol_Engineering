UDINT_TO_BYTE

Converts the value of a variable of type UDINT to the data type BYTE.

Interface definition

FUNCTION UDINT_TO_BYTE : BYTE
VAR_INPUT
    IN : UDINT;
END_VAR

Return value

Value of type BYTE
Result of the conversion.

Parameters

INPUT

IN
Value of type UDINT
Value to be converted.

Description

The function UDINT_TO_BYTE converts a value of type UDINT to a value of type BYTE.
