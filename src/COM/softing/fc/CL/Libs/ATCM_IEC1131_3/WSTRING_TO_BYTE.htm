WSTRING_TO_BYTE

Converts the value of a variable of type WSTRING to the data type BYTE.

Interface definition

FUNCTION WSTRING_TO_BYTE : BYTE
VAR_INPUT
    IN : WSTRING;
END_VAR

Return value

Value of type BYTE
Result of the conversion.

Parameters

INPUT

IN
Value of type WSTRING
Value to be converted.

Description

The function WSTRING_TO_BYTE converts a value of type WSTRING to a value of type BYTE. While converting values of type WSTRING, errors may occur. More detailed information on this subject may be found here.
