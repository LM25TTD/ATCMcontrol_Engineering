XOR

Combines the values delivered as actual parameters with bitwise exclusive OR.

Interface definition

FUNCTION XOR : ANY_BIT
VAR_INPUT 
    IN1 : ANY_BIT; 
END_VAR
VAR_IN_EXT 
    IN2 : ANY_BIT; 
END_VAR

Return value

Value of type ANY_BIT
Yields the value resulting from combining the actual parameters.

Parameters

INPUT

IN1
Value of type ANY_BIT
Value to be combined.
IN2
Value of type ANY_BIT
Value to be combined.
IN_EXT-variables are extensible. Optionally, further values can be stated to be combined.

Description

The function XOR corresponds to the operators for bitwise exclusive OR. With a bitwise exclusive OR, a bit is only set in the result if a bit is set exclusively in one of the operands that are combined. If, however, no or both bits are set at the same bit position of both operands, the result at this bit position is 0, i.e. the corresponding bit is not set. Bit masks that are combined with the output value via XOR can thus be used like switches that set bit values at specific positions to their opposite values..
