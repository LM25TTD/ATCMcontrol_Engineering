ClearLocalTaskErrno

Resets the error number of a task passed as a parameter to 0.

Interface

FUNCTION ClearLocalTaskErrno
VAR_INPUT
    NAME : ANY_STRING; 
END_VAR

Return value

None

Parameter

INPUT

NAME
Value of the type ANY_STRING
Name of the task whose error number is to be reset.
If the task name passed as the parameter does not correspond to any existing task of the current project, a runtime error is generated.

Description

ClearLocalTaskErrno resets the error number of the task passed to the NAME parameter to 0. This function should always be called before querying the error number with GetLocalTaskErrno since the error number of the calling task is not automatically reset to 0 when an error has occurred. If an error has already occurred previously in the calling task and ClearLocalTaskErrno has not been called, GetLocalTaskErrno will return the error number of the last occurred error even if no new error has occurred in a subsequent task cycle.
