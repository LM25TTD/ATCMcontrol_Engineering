DebugBreak

Interrupts the program at the position at which it is called. Functions as a breakpoint.

Interface definition

FUNCTION DebugBreak
VAR_INPUT
    IN : BOOL; 
END_VAR

Parameters

INPUT

IN
Value of type BOOL
Determines whether the program run is to be interrupted or not.
Potential values:
TRUE: The program run is interrupted.
FALSE: The program run isn´t interrupted.

Description

Has the function of a breakpoint that is set directly in the source code. Interrupts the program at the position at which DebugBreak is called and outputs a message in a dialog box indicating in which program instance the breakpoint was reached if the passed parameter is TRUE. If the passed parameter is FALSE, the function body of DebugBreak is not executed. This function has no effect if no debug connection exists.
