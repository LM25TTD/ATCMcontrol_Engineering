EnableTaskStatistic

Controls the output of status reports in the message view.

Interface definition

FUNCTION EnableTaskStatistic
VAR_INPUT
    ENABLE     : BOOL;  
END_VAR

Parameters

INPUT

ENABLE
Value of type BOOL
Determines whether the output of the task reports is to be executed or suppressed.
Potential values:
TRUE: The output of the task statistics is executed.
FALSE: The output of the task statistics is suppressed.

Description

EnableTaskStatistic controls the output of task reports in the message view of ATCM Engineering. On default, task reports are put out in the message view of ATCM Engineering after a download, a cold start or a warm start. EnableTaskStatistic can determine user defined if these messages are to be put out or suppressed.
