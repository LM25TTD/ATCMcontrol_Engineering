EnableWatchdog

Enables and disables the software watchdog for monitoring the timeout for the processing time of a program.

Interface

FUNCTION EnableWatchdog
VAR_INPUT
    ENABLE : BOOL;
END_VAR


END_FUNCTION

Return value

None

Parameter

INPUT

ENABLE
Value of the type BOOL
Enables or disables the software watchdog.
Possible values:
FALSE: Disables the software watchdog.
TRUE: Enables the software watchdog.

Description

The EnableWatchdog function is used to enable and disable the software watchdog in the 4CEC runtime system. The software watchdog monitors the execution time of a program running on an EC target. If a program needs a processing time of more than 95 ms, the calling task is stopped to prevent a timeout generated by the hardware watchdog of the bus modules.

If a program requires more than 95ms of processing time, the software watchdog can be disabled. In this case, the ECRefreshKBus function must be called before the timeout of 95 ms expires to prevent the hardware watchdog of the modules from generating a timeout.
