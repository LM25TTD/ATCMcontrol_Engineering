FC_DEBUG

Outputs a debug string in the message view and in the message history of the console.

Interface definition

FUNCTION_BLOCK FC_DEBUG
VAR_INPUT
    ENABLE     : BOOL:=TRUE;  
    STR        : WSTRING;     
END_VAR

Parameters

INPUT

ENABLE
Value of type BOOL
Starts or stops the output of a string.
Potential values:
TRUE: String is put out.
FALSE: String isn´t put out.
The standard value is TRUE.
STR
Value of type WSTRING
String, which is put out in the message view of ATCM Engineering or in the message history of the console.

Description

FC_DEBUG outputs the string passed in STR in the message view of ATCM Engineering or in the message history of the console if the input ENABLE has the value TRUE. If ENABLE has the value FALSE, nothing is put out.
