GetLocalTaskErrno

Yields the error number of the task passed as parameter.

Interface definition

FUNCTION GetLocalTaskErrno : DINT
VAR_INPUT
    NAME : ANY_STRING;
END_VAR

Return value

Value of type DINT
Error number

Parameters

INPUT

NAME
Value of type ANY_STRING
Name of the task whose error number is to be detected.

Description

GetLocalTaskState returns the error number of the task passed to NAME. If the task passed as a parameter doesn´t comply to any existing task in the current project, an exception is triggered. Possible returns:

0 No Error
2 Task is blocked
3 Runtime error in the task
