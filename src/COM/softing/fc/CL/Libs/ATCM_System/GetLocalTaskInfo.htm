GetLocalTaskInfo

Provides information on the current task status.

Interface

FUNCTION GetLocalTaskInfo
VAR_INPUT
    NAME : ANY_STRING; 
END_VAR

VAR_IN_OUT
    INFO : TASKINFO;
END_VAR

Return value

None

Parameter

INPUT

NAME
Value of the type ANY_STRING
Name of the task whose task status is to be determined.
If the task name passed as the parameter does not correspond to any existing task of the current project, a runtime error is generated.

IN_OUT

INFO
Instance of the TASKINFO
Contains the information on the task status.

Description

GetLocalTaskInfo provides information on the current task status. The infomation is stored in an instance of the TASKINFO structure which must be passed to the INFO parameter.
