GetLocalTaskState

Yields the status of the task passed as a parameter.

Interface definition

FUNCTION GetLocalTaskState : DINT
VAR_INPUT
    NAME : ANY_STRING;
END_VAR

Return value

Value of type DINT
Status of the task

Parameter

INPUT

NAME
Value of type ANY_STRING
Name of the task whose status is to be determined.

Description

GetLocalTaskState returns the status of the task passed to NAME. If the task passed as a parameter doesn´t comply to any existing task in the current project, an exception is triggered.
Possible returns:

-1   STATE_ERROR Task is in error state
 0   STATE_ONCREATION Task is newly created (e.g. in the case of an incremental download)
 1   STATE_STOPPED Task was stopped
 2   STATE_RUNNING Task is running
 3   STATE_BREAK Task was interrupted by a break point
