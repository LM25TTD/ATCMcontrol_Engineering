GetTaskErrno

Returns the last error number of the calling task.

Interface definition

FUNCTION GetTaskErrno : DINT

Return value

Value of type DINT
Last error number of the calling task.

Parameters

None

Description

GetTaskErrno returns the last error number of the calling task. Error numbers <> 0 signalize an error.

The function ClearTaskErrno should always be called before interrogating the error number with GetTaskErrno since the error number of the calling task is not automatically reset to 0 when an error has occurred. If an error has already occurred previously in the calling task and ClearTaskErrno has not been called, GetTaskErrno will return the error number of the last occurred error even if no new error has occurred in a subsequent task cycle
