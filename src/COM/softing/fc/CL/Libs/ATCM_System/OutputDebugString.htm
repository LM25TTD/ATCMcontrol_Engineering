OutputDebugString

Outputs a string in the message view or in the message history of the console.

Interface definition

FUNCTION OutputDebugString
VAR_INPUT
    STR        : ANY_STRING;     
END_VAR

Parameters

INPUT

STR
Value of type ANY_STRING
Defines the message which is put out in the message view of ATCM Engineering or in the message history of the console.

Description

OutputDebugString outputs the string passed to STR in the message view or in the message history of the console.
