SFCResetToInitialStep

Resets a SFC program section.

Interface definition

FUNCTION SFCResetToInitialStep
VAR_INPUT
    ENABLE : BOOL; 
END_VAR

Parameters

INPUT

ENABLE
Value of type BOOL
Starts the operation.
Potential values:
TRUE: Operation is started.
FALSE: Operation is not started.

Description

Resets an SFC program section if the passed parameter is TRUE; with FALSE, no reset is carried out. If an SFC program section is reset, the program evolution is restarted from the initial step. All transitions and variables are reset to their initial values. This function may only be used within POUs that contain an SFC program section. The reset triggered by this function always resets the SFC network within which the function was called
