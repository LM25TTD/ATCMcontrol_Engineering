SignalError

Transfers the current task into an error state.

Interface definition

FUNCTION_BLOCK SignalError

VAR_INPUT
    ENABLE     : BOOL:=TRUE;  
    ERRNO      : DINT:=0;     
END_VAR

Parameters

INPUT

ENABLE
Value of type BOOL
Defines, together with the parameter ERRNO, whether the body of SignalError is to be executed. ERRNO
Value of type DINT
Defines the error number to be set for the calling task.

Description

SignalError transfers the current task into an error state if ENABLE has the value TRUE and ERRNO a value <> 0. If ERRNO has the value 0, the task is not transferred into an error state, even when ENABLE has the value TRUE.The transfer of a task into an error state results in the task being stopped. The error number of the affected task is set to the value of ERRNO in that case.
