StartLocalTask

Starts the task passed as parameter.

Interface definition

FUNCTION StartLocalTask
VAR_INPUT
    NAME : ANY_STRING;
END_VAR

Parameters

INPUT

NAME
Value of type ANY_STRING
Name of the task to be started.

Description

StartLocalTask starts the task passed to NAME. If the task passed as a parameter doesn´t comply to any existing task in the current project, an exception is triggered.
