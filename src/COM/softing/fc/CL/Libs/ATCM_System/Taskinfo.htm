TaskInfo

Contains information on the task status of a task.

Interface

TYPE
     TaskInfo : STRUCT
         STATE    : DINT;
         PRIORITY : DINT;
         CYCLE    : TIME;
         ETMAX    : TIME;
         ERRNO    : DINT;
     END_STRUCT;
END_TYPE

Structure Elements

STATE

Value of the type DINT
Delivers the task status
Possible return values:
-1   STATE_ERROR Task is in error status
 0   STATE_ONCREATION Task is being newly created (e.g in the case of an incremental download)
 1   STATE_STOPPED Task was stopped
 2   STATE_RUNNING Task is running
 3   STATE_BREAK Task was interrupted by breakpoint

PRIORITY

Value of the type DINT
Delivers the task priority.

CYCLE

Value of the type TIME
Delivers the task cycle time in milliseconds.

ETMAX

Value of the type TIME
Delivers the maximum execution time in milliseconds.

ERROR

Value of the type DINT
Delivers an error number if an error has occurred.
The possible values are defined in the error constants.

Description

In an instance of the TaskInfo structure, information on the current status of a task is stored. The structure instance must be passed to the INFO parameter of the GetLocalTaskInfo function.
