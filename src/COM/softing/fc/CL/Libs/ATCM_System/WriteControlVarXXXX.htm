WriteControlVar[BOOL|DINT|LINT|LREAL|WSTRING]

Write a value to a named Control variable.

Interface

FUNCTION WriteControlVar<type> : BOOL
VAR_INPUT
   sPath : WSTRING;
   val   : <type>
END_VAR
 
<type> = BOOL, DINT, LINT, LREAL, WSTRING

Parameter

INPUT

sPath

Full qualified access path of the variable. Array elements are addressed with 0, 1, 2, ... .

 

val

The value to write.

 

Returnvalue

If no error TRUE. If any error FALSE will be returned and a corresponding  error message is written to the Control tab by using OutputDebugString

There are two kinds of errors: Firstly the access path 'sPath' may be incorrect and secondly the data type is wrong (no automatic type convertions are made).

Beispiel:

	bOK := ReadControlVarBOOL("Prog1.myFB.aBool", aBool);	
	bOK := ReadControlVarDINT("aGlobal", aDint);
	bOK := ReadControlVarLREAL("Prog1.myArray.0", aLreal);


Hinweis:  These functions should be used only rarely, they 
are not very efficient.
