getTimeSinceSystemBoot

Returns the time in milliseconds that has elapsed since the last time the system was booted on a control PC.

Interface definition

FUNCTION getTimeSinceSystemBoot : TIME

Return value

Value of type TIME
Time that elapsed since the last time the system was booted on a control PC.

Parameters

None

Description

getTimeSinceSystemBoot returns the time in milliseconds that has elapsed since the last time the system was booted on a control PC.
