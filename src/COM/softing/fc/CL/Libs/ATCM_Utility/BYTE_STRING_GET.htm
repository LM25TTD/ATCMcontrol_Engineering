BYTE_STRING_GET

Reads the value of a byte in a bytestring.

Interface definition

FUNCTION BYTE_STRING_GET : BYTE
VAR_INPUT 
    BSTR  : BYTE_STRING;
    INDEX : DINT;
END_VAR

Return value

Value of type BYTE.
Value of the byte read in the bytestring.

Parameters

INPUT

BSTR
Instance of the structure BYTE_STRING.
Bytestring, from which a value is to be read.
INDEX
Value of type DINT
Defines the index of the byte whose content is to be read out of the bytestring.

Description

Returns the value of that byte in the BSTR bytestring which is located at the byte string position defined in the index.

Before calling BYTE_STRING_GET, the instance of BYTE_STRING that is passed as a parameter must either be initialized with the function BYTE_STRING_INIT or the structure variable size of BSTR must be set to a value >= INDEX since the size of the byte string is initialized by default with the value 0 when an instance of BYTE_STRING is created. Thus, if BYTE_STRING_GET is called with INDEX > 0, this would result in an attempt to access a nonexistent index and generate a runtime error.
