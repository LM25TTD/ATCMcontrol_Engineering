BYTE_STRING_SET

Changes the value of a byte in a bytestring.

Interface definition

FUNCTION BYTE_STRING_SET
VAR_IN_OUT 
    BSTR  : BYTE_STRING; 
END_VAR
VAR_INPUT 
    INDEX : DINT;
    VALUE : BYTE;
END_VAR

Parameters

IN_OUT

BSTR
Instance of the structure BYTE_STRING.
Bytestring, in which a value is to be changed.

INPUT

INDEX
Value of type DINT.
Defines the index of the byte whose value has to be changed in the bytestring.
VALUE
Value of type BYTE.
Defines the value to be written.

Description

Writes the VALUE to that byte of the BSTR byte which is specified in the index.

Before calling BYTE_STRING_SET, the instance of BYTE_STRING that is passed as a parameter must either be initialized with the function BYTE_STRING_INIT or the structure variable size of BSTR must be set to a value >= INDEX since the size of the byte string is initialized by default with the value 0 when an instance of BYTE_STRING is created. Thus, if BYTE_STRING_GET is called with INDEX > 0, this would result in an attempt to access a nonexistent index and generate a runtime error.
