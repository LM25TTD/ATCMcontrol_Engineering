BYTE_STRING_TO_WSTRING

Converts a bytestring into a WSTRING.

Interface definition

FUNCTION BYTE_STRING_TO_WSTRING : WSTRING
VAR_INPUT 
    BSTR  : BYTE_STRING;
END_VAR

Return value

Value of type WSTRING.
Contains the bytestring converted into a WSTRING.

Parameters

INPUT

BSTR
Instance of the structure BYTE_STRING.
Bytestring that is to be converted into a WSTRING.

Description

Converts the byte string stored in the BYTE_STRING instance that is passed as a parameter into a WSTRING. The function returns the converted bytestring.

BYTE_STRING_TO_WSTRING is not suitable for reading non-ASCII characters like i.e. umlauts. Because of this, use this function only for reading out bytestrings at serial communication and at communication via UDP. For reading textfiles with function blocks of the ATCM_FileAccess library, the function COPY_WSTRING_FROM_BYTE_STRING stands by for the conversion of bytestrings to WSTRINGs.
