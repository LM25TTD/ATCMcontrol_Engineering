BYTE_SWAP_WORD

Converts the value passed from the Big-Endian- to the Little-Endian format and vice versa.

Interface definition

FUNCTION BYTE_SWAP_WORD : WORD
VAR_INPUT
    VALUE : WORD;
END_VAR

Return value

Value of type WORD
Converted value of VALUE.

Parameters

INPUT

VALUE
Value of type WORD
Specifies the value to be converted.

Description

BYTE_SWAP_WORD reverses the byte order of VALUE and returns the result. The purpose of this operation is to convert the value passed from the Big-Endian- to the Little-Endian format and vice versa.
