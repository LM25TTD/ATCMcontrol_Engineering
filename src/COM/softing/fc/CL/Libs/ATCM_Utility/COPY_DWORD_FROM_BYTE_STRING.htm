COPY_DWORD_FROM_BYTE_STRING

Reads a value of type DWORD at a specified position within a bytestring.

Interface definition

FUNCTION COPY_DWORD_FROM_BYTE_STRING : DWORD
    VAR_IN_OUT
        POS : DINT;
        BSTR : BYTE_STRING;
    END_VAR

Return value

Value of type DWORD
Value that has been read out of the bytestring.

Parameters

IN_OUT

POS
Value of type DINT
Specifies the position in the bytestring from which the value is to be read. Counting starts from 0 on for the first position within the bytestring and is carried out in the unit bytes.
After the function has been carried out successfully, POS is overwritten with the value of the first position in the bytestring that follows the last byte that has been read. If the operation was a failure, POS has the value -1.
BSTR
Instance of the structure BYTE_STRING
Bytestring from which a value is to be read.

Description

COPY_DWORD_FROM_BYTE_STRING reads a value of type DWORD from the position passed in POS on from the bytestring passed in BSTR and returns this value.
