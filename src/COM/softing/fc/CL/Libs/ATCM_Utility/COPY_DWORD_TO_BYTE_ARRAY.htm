COPY_DWORD_TO_BYTE_ARRAY

Writes a value of the type DWORD at a specific position within a byte array.

Interface

FUNCTION COPY_DWORD_TO_BYTE_ARRAY
VAR_INPUT
    VALUE : DWORD;
END_VAR
VAR_IN_OUT
    POS : DINT;                                 
    BYTES : ARRAY OF BYTE;
END_VAR

Parameter

INPUT

VALUE
Value of the type DWORD
Value to be inserted at the byte array position passed in POS. Counting starts from 0 for the first position in the byte array and is performed in byte units.

IN_OUT

POS
Value of the type DINT
Specifies at which position in the byte array the value passed to VALUE is to be inserted. Counting starts from 0 for the first position in the byte array and is performed in byte units.
After the successful execution of the function, POS is overwritten by the value of the first byte array position that follows the last byte written. If the execution of the function failed, POS receives the value -1.
BYTES
Array of the type BYTE
Byte array in which a value is to be written.

Description

COPY_DWORD_TO_BYTE_ARRAY writes a value of the type DWORD in the byte array passed to BYTES at the position passed to POS. Values that are located in the byte array at the position specified in POS are overwritten by the value defined in VALUE.
