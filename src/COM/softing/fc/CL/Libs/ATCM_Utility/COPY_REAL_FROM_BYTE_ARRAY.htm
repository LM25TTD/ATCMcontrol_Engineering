COPY_REAL_FROM_BYTE_ARRAY

Reads a value of the type REAL at a specific position within a byte array.

Interface

FUNCTION COPY_REAL_FROM_BYTE_ARRAY : REAL
VAR_IN_OUT
    POS : DINT;
    BYTES : ARRAY OF BYTE;
END_VAR

Return value

Value of the type REAL
Value that was read from the byte array.

Parameter

IN_OUT

POS
Value of the type DINT
Specifies as from which position in the byte array the value is to be read. Counting starts from 0 for the first position in the byte array and is performed in byte units.
After the successful execution of the function, POS is overwritten by the value of the first byte array position that follows the last byte read. If the execution of the function failed, POS receives the value -1.
BYTES
Instance of an array of byte.
Byte array from which a value is to be read.

Description

Starting from the position passed to POS, COPY_REAL_FROM_BYTE_ARRAY reads a value of the type REAL from the byte array passed to BSTR and delivers this value as the return value.
