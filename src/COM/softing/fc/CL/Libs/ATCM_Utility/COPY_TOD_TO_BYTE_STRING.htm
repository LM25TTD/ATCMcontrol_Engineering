COPY_TOD_TO_BYTE_STRING

Writes a value of type TIME_OF_DAY to a specified position within a bytestring.

Interface definition

FUNCTION COPY_TOD_TO_BYTE_STRING
    VAR_INPUT
        VALUE : TIME_OF_DAY;
    END_VAR
    VAR_IN_OUT
        POS : DINT;
        BSTR : BYTE_STRING;
    END_VAR

Parameters

INPUT

VALUE
Value of type TIME_OF_DAY
Value to be inserted into the bytestring at the position passed in POS.

IN_OUT

POS
Value of type DINT
Specifies the position where the value passed in VALUE is to be inserted. Counting starts from 0 on for the first position within the bytestring and is carried out in the unit bytes.
After the operation has been carried out successfully, POS is overwritten with the value of the first position in the bytestring that follows the last byte written. If the operation was a failure, POS has the value -1.
BSTR
Instance of the structure BYTE_STRING
Bytestring to which a value is to be written.

Description

COPY_TOD_TO_BYTE_STRING writes at the position passed in POS a value of type TIME_OF_DAY into the bytestring passed to BSTR . Values located in the bytestring at the position specified in POS are overwritten with the value specified in VALUE.

If the bytestring passed is to small to incorporate the value passed in VALUE, or if the position, at which the value is to be inserted, is greater than the actual length of the bytestring, the bytestring is extended to the minimal size needed.
