UDP-Error constants

Error constants for data transfer errors with UDP.

Constant
Numeric value
 Description
UDP_ERR_SOCK_INIT_FAILED -201 The socked couldn´t be initialized.
UDP_ERR_SOCK_NOT_OPENED -202 The socket wasn´t initialized before the attempt to send or recieve data.
UDP_ERR_UNKNOWNHOST -203 The given host name could not be found.
UDP_ERR_SEND -204 An IO-Error occurred while trying to send a message.
UDP_ERR_RECV -205 An IO-Error occurred while trying to receive a message.
UDP_ERR_RECV_MSG_LONGER -206 The packet read contains more bytes then defined in the parameter DATA_LEN. The bytes exceeding the value defined in DATA_LEN are cut off.
UDP_ERR_SEND_MSG_TOO_LONG -207 The message is too long to be transported via a UDP socket.
UDP_ERR_SEND_HOST_UNREACH -208 The specified communication partner can not be contacted at the moment.
UDP_ERR_LOOKUP_RUNNING -209 A parameter passed to an instance of the asyncronic funcion block UDP_LOOKUP has been changed, though the operation carried by this instance is not completed. In case of this error, the parameters changed during the execution of the operation are ignored. The operation is carried on with the parameters passed first.
UDP_ERR_LOOKUP_HN_CHANGE -210 The host name passed as parameter to an instance of the asyncronic function block UDP_LOOKUP has been changed after the operation carried out by the instance has been completed, even though no new search for an IP-adress has been started with a raising edge at the paramter EN_L. This error value is to indicate that the value delivered at the output HOST_ADDRESS is not the IP-adress of the host actually passed to the parameter HOST_NAME but the IP-adress of the host that has been found after the completion of the last search operation that wascarried out.
UDP_ERR_PORTNR_OUTOFRANGE -212 The port number passed as parameter is not within the valid range of 0 to 65535.

Description

The DP-error constants define the value for the element LastErrorNr of the structure UDP_SOCKET_HANDLE und the parameter ERROR_NR of the UDP function blocks.
