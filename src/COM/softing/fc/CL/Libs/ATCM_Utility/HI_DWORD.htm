HI_DWORD

Extracts from the higher significant proportion of a value of type LWORD a new value of type DWORD.

Interface definition

FUNCTION HI_DWORD : DWORD
    VAR_INPUT
        VALUE : LWORD;
    END_VAR

Return value

Value of type DWORD
Value generated from the higher significant proportion of VALUE.

Parameters

INPUT

VALUE
Value of type LWORD
Value from which the higher significant proportion is to be extracted.

Description

HI_DWORD extracts a new value of type DWORD from the higher significant proportion of the value passed to VALUE and returns the result.
