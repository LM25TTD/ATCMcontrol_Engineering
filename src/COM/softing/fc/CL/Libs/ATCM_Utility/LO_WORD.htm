LO_WORD

Extracts from the lower significant proportion of a value of type DWORD a new value of type WORD.

Interface definition

FUNCTION LO_WORD : WORD
    VAR_INPUT
        VALUE : DWORD;
    END_VAR

Return value

Value of type WORD
Value generated from the lower significant proportion of VALUE.

Parameters

INPUT

VALUE
Value of type DWORD
Value from which the lower significant proportion is to be extracted.

Description

LO_WORD extracts a new value of type WORD from the lower significant proportion of the value passed to VALUE and returns the result.
