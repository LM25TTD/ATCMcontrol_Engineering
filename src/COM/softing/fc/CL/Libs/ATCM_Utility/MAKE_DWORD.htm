MAKE_DWORD

Generates a new unsigned integer value of type DWORD from two values of type WORD.

Interface definition

FUNCTION MAKE_DWORD : DWORD
    VAR_INPUT
        LO_WORD : WORD;
        HI_WORD : WORD;
    END_VAR

Return value

Value of type DWORD
Value generated from LO_WORD and HI_WORD.

Parameters

INPUT

LO_WORD
Value of type WORD
Lower significant proportion of the value to be generated.
HI_WORD
Value of type WORD
Higher significant proportion of the value to be generated.

Description

MAKE_DWORD combines the integer values passed to LO_WORD and HI_WORD to a new unsigned integer value with the amount of bytes doubled and returns the result. The value in LO_WORD defines the lower siginificant and the value in HI_WORD the higher significant proportion of the new value to be generated.
