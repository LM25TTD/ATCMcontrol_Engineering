Registration of ATCMControl Embedded V2.1

After installation, all installed ATCMControl Embedded components are running in demo mode. This mode involves particular restrictions depending on the components you have installed. For instance, you cannot access the I/O addresses of field devices in demo mode. To use ATCMControl Embedded without restrictions, you need to register the installed components. Registering is machine-dependent. This means that the registration key created by ATCMControl Embedded for enabling the components is only valid on the PC for which you have requested the registration key.

There are two ways to register your ATCMControl Embedded components:

Temporary registration:
Select this registration type to test ATCMControl Embedded for a specific time without restrictions of functionality. After a specific period of time, the registration key sent to you by ATCMControl expires and ATCMControl Embedded runs in demo mode again.

Permanent registration:
If you have decided to use ATCMControl Embedded in your specific area of application in the long term, select this registration type. The registered components can be used without restrictions of functionality and without a time limit.

Click here to go to the description of the registering procedure.

Apart from a machine-dependent registering of ATCMControl Embedded components, you can also obtain a hard key for machine-independent registering. The hard key can be connected to the USB port. The ATCMControl Embedded components which you have installed in that PC and for which the hard key has been registered are then enabled for unlimited use. With the hard key, you can enable ATCMControl Embedded installations for unlimited use alternately on different PCs.

To order a hard key, you do not have to create an installation key as described in the ATCMControl Embedded Registration section. Indicate the order number and license number in your order and send it to support@4control.de.
