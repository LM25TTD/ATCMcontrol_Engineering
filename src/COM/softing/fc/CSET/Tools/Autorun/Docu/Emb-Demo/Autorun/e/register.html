ATCMControl Embedded V2.1 Registration
  1. Call the Softing License Manager by clicking Programs \Softing AG \Softing License Manager in the Windows start menu.
  2. In the Product Name column, the Softing License Manager lists all ATCMControl Embedded components that can be installed. The other columns display information on the license and installation states:

To register your ATCMControl Embedded components, click Create Installationkey in the Softing License Manager.

  1. In the next dialog box, enter the host name of the computer in which the components to be registered have been installed and enter your company name. These data are required for creating the installation key. In the Machinename field, the host name of the computer on which you have started the Softing License Manager is displayed by default; the Company field shows the company name found in the Windows registry database. If you want to register ATCMControl Embedded components which are installed in a different computer, you can overwrite these fields. The permitted number of characters is limited to a total of 24 characters for both field.



Click Continue to confirm your entries.

  1. The entries are displayed again in the next dialog box. Check whether they are correct and then click OK.



    To correct the entries, click Retry. This takes you back to the previous dialog box.

  1. When you have confirmed your entries with Continue, an installation key is created for the computer with the specified host name. Send this key to ATCMControl to have the final registration key generated for you.

  2. To do this, click Copy Installationkey to Clipboard to copy the installation key to the Windows clipboard. You can now paste the installation key from the clipboard into Notepad or into your Email program.

  3. Send the installation key together with the order number and license number to support@4control.de. The order number and license number are essential to be able to send you a final registration key as soon as possible.

  4. You will receive the final registration key by Email as soon as possible. When you have received the final registration key, open the Softing License Manager again and enter the key in the Enter Registration Key(s) field.

  5. Then click Enable. The registered ATCMControl Embedded components are now registered and the license state is displayed in the list of ATCMControl Embedded components. You can now use the registered ATCMControl Embedded components without restrictions.
