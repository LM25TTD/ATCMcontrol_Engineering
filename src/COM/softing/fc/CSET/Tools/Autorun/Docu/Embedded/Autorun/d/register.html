ATCMControl Embedded V2.1 lizensieren
  1. Rufen Sie den Softing License Manager auf, indem Sie im Windows Start-Menü auf Programme \Softing AG \Softing License Manager klicken.
  2. Der Softing License Manager Ihnen in der Spalte Product Name alle installierbaren ATCMControl Embedded-Komponenten an. In den weiteren Spalten werden Informationen zum Lizenz- und Installationsstatus angezeigt:

Um Ihre ATCMControl Embedded-Komponenten zu lizensieren, klicken sie im Softing License Manager auf Erzeuge Installkey.

  1. Im nun folgenden Dialog geben Sie den Hostnamen des Rechners, auf dem die zu lizensierenden Komponenten installiert sind, und den Namen Ihrer Firma ein. Diese Angaben werden für die Erzeugung des Installationssschlüssels benötigt. Im Feld Arbeitsplatz wird standardmäßig der Hostname des Rechners, auf dem Sie den Softing License Manager gestartet haben, und im Feld Firma der in der Windows-Registrierungsdatenbank gefundene Name Ihrer Firma angezeigt. Wenn Sie ATCMControl Embedded-Komponenten, die auf einem anderen Rechner installiert sind, lizensieren wollen, können Sie diese Felder überschreiben. Die Länge der zulässigen Zeichen in beiden Felder zusammen ist auf 24 begrenzt.



Klicken Sie auf Weiter, um die Angaben zu bestätigen.

  1. In einem weiteren Dialog werden noch einmal die Angaben im vorhergehenden Dialog angezeigt. Vergewissern Sie sich, daß diese Angaben korrekt sind, und klicken Sie auf OK.



    Wollen Sie die Angaben noch korrigieren, klicken Sie auf Wiederholung. Sie gelangen dann wieder zum vorigen Dialog.

  1. Nachdem Sie die Angaben mit Weiter bestätigt haben, wird ein Installationsschlüssel für den Rechner mit dem angegebenen Hostnamen erzeugt. Diesen Schlüssel schicken Sie an ATCMControl, damit dort der endgültige Registrierschlüssel erzeugt werden kann.
  2. Klicken Sie dazu auf Installationkey in Zwischenablage kopieren, um den Installationsschlüssel in die Windows-Zwischenablage zu kopieren. Der Installationsschlüssel kann nun aus der Windows-Zwischenanlage in Notepad oder in Ihr Mailprogramm eingefügt werden.
  3. Senden Sie den Installationsschlüssel zusammen mit der Bestellnummer und der Lizenznummer an support@4control.de. Die Angabe der Bestellnummer und der Lizenznummer sind unbedingt erforderlich, damit ihnen möglichst schnell ein endgültiger Registrierschlüssel zugesandt werden kann.
  4. Sie bekommen den endgültigen Registrierschlüssel schnellstmöglich per E-Mail. Wenn Sie den endgültigen Registrierschlüssel erhalten haben, öffnen Sie wieder den Softing License Manager und geben den Schlüssel im Feld Registrierschlüssel eingeben ein.
  5. Klicken Sie dann auf Enable. Die lizensierten ATCMControl Embedded-Komponenten werden nun registriert, und der Lizensierungsstatus in der Liste der ATCMControl-Komponenten angezeigt. Sie können jetzt mit den lizensierten ATCMControl Embedded-Komponenten ohne Einschränkungen arbeiten.
