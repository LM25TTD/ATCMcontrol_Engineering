Bevor Sie die Software installieren, lesen Sie bitte den folgenden Abschnitt sorgfältig durch:
  • Diese Installationsanleitungen setzen das Vorhandensein eines Netzwerks voraus. Falls Ihr Rechner noch nicht in ein Netzwerk eingebunden ist, müssen Sie zunächst eine Netzwerkkarte einbauen, bevor Sie mit der Installation aller benötigten Komponenten für ATCMControl beginnen können.
  • Die Installation von ATCMControl setzt voraus, daß Sie sich auf Ihrem Rechner mit Windows Administratorrechten angemeldet haben. Falls Sie nicht über diese Rechte verfügen, wenden Sie sich bitte an Ihren zuständigen Systemadministrator.
  • Es empfiehlt sich, alle Vorgängerversionen von ATCMControl zu deinstallieren, bevor Sie eine neue Version installieren. Den Deinstallationsvorgang starten Sie mit der Menüoption Uninstall ATCMControl in der ATCMControl-Programmgruppe.
  • Wenn Sie mit einer früheren ATCMControl-Version mitgelieferte PROFIbus-Treiber installiert haben, wird es dringend empfohlen, diese Treiber vollständig aus Ihrem System zu entfernen, bevor Sie das Setup-Programm von ATCMControl starten. Die PROFIbus-Treiber werden bei der Installation von ATCMControl mitinstalliert. Wenn daher zuvor installierte ATCMControl PROFIbus-Treiber noch auf Ihrem System vorhanden sind, werden beim ATCMControl-Setup wahrscheinlich Konflikte mit der neuen Treiberinstallation auftreten. Nach der Deinstallation der PROFIbus-Treiber müssen Sie Ihren PC neu starten. Nach dem Neustart können Sie mit der Installation von ATCMControl beginnen.

WICHTIGER HINWEIS:

In einigen Fällen, besonders wenn Sie ATCMControl auf einem vollständig neu installierten System installieren, meldet Ihnen ATCMControl evtl., daß einige System-DLLs ersetzt wurden, und fordert Sie auf, den PC neu zu starten. Nach dem Neustart erscheint wieder der ATCMControl Installation Wizard und signalisiert durch den Eintrag 'Install ATCMControl', daß ATCMControl noch nicht installiert wurde. Der Grund für dieses Verhalten ist, daß die Installation aufgrund der Notwendigkeit eines Neustarts des PCs nach dem Ersetzen der System-DLLs nicht abgeschlossen werden konnte. Um die Installation fertigzustellen, müssen Sie den Installationsvorgang durch Klicken auf die Schaltfläche Install Next neu starten. Leider müssen Sie dazu die Lizenzvereinbarung erneut akzeptieren und den Namen Ihrer Firma sowie einige anderen Angaben nochmals eingeben. Danach wird die Installation der noch fehlenden Teile von ATCMControl fortgesetzt und der Installationsvorgang ordnungsgemäß abgeschlossen.
