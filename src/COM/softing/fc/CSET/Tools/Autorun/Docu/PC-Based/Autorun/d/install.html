Hinweise zur Installation:

ATCMControl erfordert ein bestehendes Netzwerk. Falls Ihr Rechner noch nicht in ein Netzwerk eingebunden ist, müssen Sie zunächst eine Netzwerkkarte einbauen, bevor Sie mit der Installation aller benötigten Komponenten für ATCMControl beginnen können. Wenn Sie ATCMControl nicht in einem Netzwerk betreiben möchten, müssen Sie einen sogenannten Loopback-Adapter installieren, um einen einwandfreien Betrieb zu gewährleisten.

Um ATCMControl installieren zu können, müssen Sie im System als Administratror angemeldet sein.

ATCMControl benötigt zum korrekten Betrieb unter Windows 2000/XP die MS Internet Information Services. Bei der Installation wird Ihr Betriebssystem nach diesen Komponenten durchsucht. Haben Sie diese Komponenten noch nicht installiert, werden Sie von der Informationsroutine dazu aufgefordert, die entsprechende Komponente von Ihrer Windows Installations-CD zu installieren. Halten Sie daher die Windows Installations-CD bereit.

  1. Schließen Sie alle geöffneten Anwendungen.

  2. Starten Sie Setup.exe von der ATCMControl-Installations-CD.

  3. Der Setup-Startdialog wird angezeigt.

    Klicken Sie auf OK, um mit der Installation fortzufahren.

  4. Wählen Sie das Zielverzeichnis für die Installation aus.

    Wenn Sie noch keine ATCMControl-Version installiert haben, wird als Standardpfad in der Regel C:\Programme\4ControlV2 vorgeschlagen, andernfalls der Pfad Ihrer vorhergehenden ATCMControl-Version. Um das Installationsverzeichnis zu ändern klicken Sie auf Verzeichnis ändern. In einem Verzeichnis-Auswahldialog können Sie ein anderes Verzeichnis einstellen. Klicken auf Weiter, um das Zielverzeichnis für die Installation zu bestätigen.

  5. Wählen Sie nun aus, ob Sie eine Standard- oder eine benutzerdefinierte Installation durchführen wollen. Der Unterschied zwischen beiden Installationsarten besteht darin, daß bei der Standard-Installation ein PC Target nur zusammen mit ATCM Console und ein Ethernet Controller Target nur zusammen mit dem OPC Server installiert werden kann. Bei der benutzerdefinierten Installation können diese Komponenten dagegen separat installiert werden.


    Klicken Sie auf Standard, um mit der Standard-Installation, oder auf Benutzerdefiniert, um mit der benutzerdefinierten Installation fortzufahren.

  6. Im folgenden Dialog wählen Sie im Register Setup die Komponenten aus, die Sie installieren wollen. Standardmäßig sind alle Komponenten für die Installation markiert. Um eine Komponente abzuwählen, klicken Sie auf das entsprechende Kontrollkästchen, um die Markierung zu entfernen. Per Mausklick auf ein demarkiertes Kontrollkästchen fügen sie eine Markierung wieder hinzu.


Im Bereich PC-Target können Sie folgende Komponenten auswählen:

  • ATCM PC-based Engineering
    Installiert ATCM Engineering mit den Komponenten und Standardbibliotheken zur Entwicklung von Laufzeitprogrammen für ein PC-Target.

  • ATCM PC-based Control
    Installiert in der Standard-Installation das Laufzeitsystem (ATCM Control) für die Ausführung von Laufzeitprogrammen auf einem PC-Target zusammen mit ATCM Console. In der benutzerdefinierten Installation können Laufzeitsystem und ATCM Console getrennt installiert werden.

  • Bibliotheken
    Installiert die im Listenfeld unterhalb dieses Kontrollkästchens aufgeführten Zusatzbibliotheken. Ist das Kontrollkästchen markiert, können die Zusatzbibliotheken im Listenfeld einzeln für die Installation ausgewählt werden. Wird das Listenfeld demarkiert, werden auch alle Markierungen der Bibliotheken im Listenfeld entfernt.

Im Bereich Ethernet Controller Target können Sie folgende Komponenten auswählen:

In der Standard-Installation installieren Sie bei markiertem Kontrollkästchen ATCM Ethernet Controller Engineering ATCM Engineering einschließlich der Komponenten und Standardbibliotheken zur Entwicklung von Laufzeitprogrammen für ein Ethernet Controller Target zusammen mit dem OPC-Server. Dieser wird zur Kommunikation von auf Ethernet Controllern ausgeführten Laufzeitprogrammen mit ATCM Engineering benötigt. In der benutzerdefinerten Installation können ATCM Engineering und OPC-Server getrennt installiert werden.

Die Sprachversion der installierten Komponenten nehmen Sie über die Optionsfelder Deutsch und English vor. Die Auswahl der Sprache entscheidet u.a. darüber, welche Sprachversion der Online-Hilfe zu den ausgewählten Komponenten installiert wird.

ATCMControl benötigt zur korrekten Ausführung einige Systemkomponenten und je nach Windows-Version ein entsprechendes Servicepack. Bei der Ausführung der Installation wird Ihr System nach diesen Komponenten durchsucht. Werden die erforderlichen Komponenten nicht gefunden, werden Sie während des Installationsvorgangs von der ATCMControl-CD installiert. Über die von ATCMControl benötigten Komponenten und die Version der Komponenten, die aktuell auf Ihrem System installiert sind, können Sie sich in der Registerkarte Info informieren.


Um die ausgewählten Komponenten zu installieren, klicken Sie in der Registerkarte Setup auf Installation starten.
  1. Lesen Sie die Lizenzvereinbarung sorgfältig durch, markieren Sie das Optionsfeld Ich aktzeptiere die Bedingungen der Lizenzvereinbarung, und klicken Sie auf Weiter, um mit der Installation fortzufahren.

  1. Bestätigen Sie die Frage im folgenden Dialog, ob Sie mit der Installation fortfahren wollen, mit Ja.

  1. Vor der Installation der ausgewählten Komponenten werden gegebenenfalls die benötigten Systemkomponenten installiert. Ein Dialog informiert Sie über den Fortschritt der Installation und die Komponenten, die noch installiert werden müssen.

Während der Installation von System- und ATCMControl-Komponenten kann es mehrfach vorkommen, daß Sie zum Neustart von Windows aufgefordert werden. Folgen Sie in diesem Fall den Anweisungen auf Ihrem Bildschirm und starten Sie Windows neu. Nach dem Neustart wird das ATCMControl-Setup automatisch wieder aufgerufen und fährt mit der Installation fort.

Hinweis:

Wenn Sie unter Windows 2000 / Windows XP die MS Internet Information Services noch nicht installiert haben, wird zuerst eine HTML-Seite angezeigt. Diese Seite fordert Sie auf, die Internet Information Services zu installieren. Folgen Sie den Anweisungen auf dieser Seite. Sie können mit der weiteren Installation erst dann fortfahren, wenn Sie die entsprechenden Services installiert haben.

Falls Sie I/Os von PROFIbus DP-Feldgeräten in Ihren Programmen verwenden wollen, müssen Sie nach der Installation von ATCMControl Ihre PROFIBus-Treiber konfigurieren.
