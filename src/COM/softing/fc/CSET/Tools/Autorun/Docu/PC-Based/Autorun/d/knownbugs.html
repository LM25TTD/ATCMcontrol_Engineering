Anmerkungen zu dieser Version

In diesem Kapitel werden einige Probleme aufgeführt, die bei der Arbeit mit ATCMControl auftreten können, und Maßnahmen aufgezeigt, wie diese Probleme vermieden bzw. gelöst werden können.
Problembeschreibung Lösung
Die erneute Installation der Peer Web Services führt zu einer Änderung der ASP-Installation. Infolge dessen kann ATCM Console die ASP-Seiten nicht mehr korrekt anzeigen. Installieren Sie die ASP-Unterstützung neu.
Wird ein Pipe-Symbol (|) in einer WSTRING-Variable verwendet, wird dieses Zeichen bei der Beobachtung des Strings in ATCM Engineering oder ATCM Console als Leerzeichen angezeigt. Verwenden Sie keine Pipe-Symbole (|) für den Inhalt von Variablen.
Wenn Sie mit einem sehr großen Projekt (mit über 5000 Variablen) arbeiten, kann der Online-Change lange dauern, so daß das System scheinbar nicht mehr reagiert. Dies stellt kein Problem dar, sondern dauert einfach einige Zeit. Sie können diese Zeit verringern, indem Sie den Systemspeicher auf Ihrem ATCM Control-PC erhöhen.
Der Menübefehl "Drucken" in ATCM Console (Internet Explorer 4) funktioniert nicht. Diese Funktion wird nicht unterstützt. Wenn Sie in ATCM Console ausdrucken möchten, verwenden Sie die Windows-Kopierfunktion.
Wenn ein Projekt auf ATCM Control geladen wird, während der PROFIbus-Konfigurator offen ist, kann das Download nicht erfolgreich abgeschlossen werden. Schließen Sie den PROFIbus-Konfigurator, bevor Sie mit dem Download eines Projekts beginnen.
Selbst wenn die Ressource gestoppt wurde, wird die Ausführung bestehender RTCE-Tasks gestartet, wenn der DP-Konfigurator geöffnet wird und die Konfiguration auf ein ATCM Control geladen wird. Falls Sie RTCE-Tasks in Ihrem Projekt verwenden, sollten Sie den DP-Konfigurator nur starten, wenn gegenwärtig kein Download des Projekts durchgeführt wird.
Beim Versuch, mehr als eine Instanz von ATCM Console zu öffnen kann u. U. eine Fehlermeldung des Typs "Proxy error.." auftreten. Zur Vermeidung dieses Problems, sollten Sie die Option Browserfenster in einem eigenen Prozess öffnen auf der Registerkarte Erweitert in den Internetoptionen Ihres Internet Explorers auswählen.
Unter Umständen können Skriptfehler beim Laden oder Ausführen eines Projekts auf ATCM Console auftreten. Die Skriptfehler treten nicht auf, wenn Sie die neueste Version des Internet Explorers verwenden, die auf der ATCMControl-CD enthalten ist. Installieren Sie diese Version, um Skriptfehler zu vermeiden.
