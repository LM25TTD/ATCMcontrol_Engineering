Before installing the software, please read the following information thoroughly from beginning to end:
  • These installation instructions require an existing network. If your computer is not integrated into a network, you have to install a network adapter first before you start installing all necessary components for ATCMControl.
  • Installing ATCMControl requires that you have logged on to your computer with Windows administrator rights. If you do not have these rights, please contact your local system administrator.
  • It is recommended that you uninstall all previous versions of ATCMControl before installing this new version. The uninstallation procedure can be started by selecting the Uninstall ATCMControl menu option in the ATCMControl program group.
  • If you have installed any PROFIbus drivers delivered with an earlier version of ATCMControl, it is strongly recommended that you remove all of these drivers from your system before starting the setup program of ATCMControl. The installation of PROFIbus drivers is part of in the installation routine of ATCMControl. Leaving any previously installed ATCMControl PROFIbus drivers on your system will therefore quite probably cause conflicts with the new driver installation by the ATCMControl setup. After having uninstalled the PROFIbus drivers, you need to restart your computer. After restarting, you can start installing ATCMControl.

Important Note:

In some cases, especially when installing ATCMControl on a freshly installed system, ATCMControl may inform you that some system DLLs have been replaced and asks you to restart your computer. After restart, the Installation Wizard of ATCMControl appears again and indicates by the item 'Install ATCMControl' that ATCMControl has not yet been installed. The reason for this behavior is that the installation could not be completed because of the necessity of restarting the computer after replacing the system DLLs. To complete the installation, you have to start the installation process again by clicking the Install Next button. Unfortunately you need to confirm your agreement to the license terms again and reenter the name of your company and some other required information. After having done this, the installation of the still missing parts of ATCMControl is continued and the installation process is properly completed.
