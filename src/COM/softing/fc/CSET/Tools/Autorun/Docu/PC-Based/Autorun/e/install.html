Installation notes:

ATCMControl requires an installed network. If your computer is not integrated into a network, you have to install a network adapter first before you start installing all necessary components of ATCMControl. If you do not want to operate ATCMControl within a network, you have to install the so called loopback adapter for correct operation.

Before you install ATCMControl, you need to log on to the system with administrator rights.

For proper operation, ATCMControl requires the MS Internet Information Services for operation with Windows 2000/XP. The installation program will search for these components in your operating system. If you have not yet installed these components, the installation routine will ask you to install the corresponding components from your Windows Installation CD. You may therefore want to keep the Windows Installation CD ready.

  1. Close any applications you have running.

  2. Start Setup.exe from the ATCMControl Installation CD.

  3. The Setup startup dialog box is displayed.

    Click OK to continue installation.

  4. Select the destination directory for the installation.
    If this is the first version of ATCMControl you are installing, the setup wizard usually offers the default path C:\Programs\4ControlV2; otherwise, the path of your previous ATCMControl version will be offered. To change the installation directory, click Change Directory. A directory selection dialog box opens in which you can choose a different directory. Click Continue to confirm the destination directory for the installation.

  5. Next, select either the standard or the user-defined installation. The difference between the two installation types is that, in the standard installation, a PC target can only be installed together with ATCM Console and an Ethernet Controller can only be installed together with the OPC Server. In the user-defined installation, these components can be installed separately.
    Click Standard to proceed with the standard installation or click Expert to proceed with the user-defined installation.

  6. On the Setup tab in the next dialog box, select the components you want to install. By default, all components are selected for installation. To deselect a component, click the corresponding check box to remove the check mark. To select a deselected check box, simply click it again.


In the PC Target group box, you can select the following components:

  • ATCM PC-based Engineering
    Installs ATCM Engineering with the components and standard libraries for the development of runtime programs for a PC target.

  • ATCM PC-based Control
    Installs the runtime system (ATCM Control) for the execution of runtime programs on a PC target; in the standard installation, the runtime system is installed together with ATCM Console. In the user-defined installation, they can be installed separately.

  • Libraries
    Installs the additional libraries displayed in the list box below this check box. When the check box is enabled, the additional libraries shown in the list box can be selected individually for installation. When the check box is disabled, all libraries in the list box are deselected.

In the Ethernet Controller Target group box, you can select the following components:

When you enable the ATCM Ethernet Controller Engineering check box in the standard installation, ATCM Engineering as well as the components and standard libraries for the development of runtime programs for an Ethernet Controller Target are installed together with the OPC Server. The OPC Server is required for communication between the runtime programs executed on the Ethernet Controllers and ATCM Engineering. In the user-defined installation, ATCM Engineering and the OPC-Server can be installed separately.

For proper operation, ATCMControl requires particular system components and a correspon-ding service pack, depending on the Windows version you are using. The installation program will search for these components in your operating system during the setup procedure. If the required components are not found, they are installed during installation from the ATCMControl CD. On the Info tab, you will find information on the individual components required by ATCMControl and on the versions of the components that are currently installed on your system.

To install the selected components, click Start Installation on the Setup tab.
  1. Read the license agreement thoroughly from beginning to end, select the radio button
    I accept the terms in the license agreement, and click Continue to proceed.
  1. In the next dialog box, answer the query whether you want to continue with the setup with Yes.

  1. Before installing the selected components, the required system components will be installed, if necessary. A dialog box informs you about the progress of installation and shows you which components still need to be installed.


During the installation of system and ATCMControl components, you may be asked a few times to restart Windows. In this case, follow the instructions on the screen and restart Windows. After the restart, the ATCMControl Setup is opened automatically and continues the installation.

Note:

If you have not yet installed the MS Internet Information Services under Windows 2000/XP, an HTML page will be displayed first. This page will ask you to install the Internet Information Services. Follow the instructions given on this page. You can only continue the setup procedure if you have installed the corresponding services.

If you want to use the I/Os of PROFIbus DP field devices in your programs, you need to configure the PROFIBus drivers after the installation of ATCMControl.
