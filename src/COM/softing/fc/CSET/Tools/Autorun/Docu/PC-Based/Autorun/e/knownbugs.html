Release Notes

This chapter lists some problems wich in some cases may occur while working with ATCMControl and presents measures to avoid or solve these problems.
Problem Description Solution
Reinstallation of Peer Web Services changes ASP installation. Consequence is that ATCM Console cannot display ASP pages correctly anymore. Reinstall ASP support.
If a Pipe (|) symbol is used in a WSTRING variable this symbol is displayed as blank when the string is watched in ATCM Engineering or ATCM Console. Do not use the Pipe (|) symbol for variable contents.
In case of working with a very large project (more than 5000 variables) the the online change could last long and the system seems to be not responding. There is no problem, it just takes some time. You can reduce this time by increasing your system memory on your ATCM Control PC.
Menu command "Print" of ATCM Console (Internet Explorer 4) does not work. This function isn't supported. If you want to make a ATCM Console print, use the Windows hard copy function.
If a project is downloaded on ATCM Control while the PROFIbus configurator is opened, the download process fails. Close the PROFIbus configurator, before you begin with the download of a project.
Even if the ressource is stopped any existing RTCE tasks will start running when the DP configurator is opened and the configuration is downloaded on ATCM Control. In case you are using RTCE tasks in your project, you should start the DP configurator only when the project isn't currently downloaded.
When trying to open more then one instance of ATCM Console in some cases you may get an error of the kind "Proxy error..". To avoid this problem you should select Launch browser windows in a separate process in the Advanced-tab of the Internet Options in your Internet Explorer.
In some cases script errors may occur when loading or performing a project on ATCM Console. The script errors do not occur with the newest release of the Internet Explorer which is contained in the ATCMControl CD. Install this version to avoid script errors.
