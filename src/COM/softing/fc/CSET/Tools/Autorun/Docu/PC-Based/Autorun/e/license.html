Registration of ATCMControl

After installation, all installed ATCMControl components are running in demo mode. This mode involves particular restrictions depending on the components you have installed. For instance, you cannot access the I/O addresses of field devices in demo mode. To use ATCMControl without restrictions, you need to register the installed components. Registering is machine-dependent. This means that the registration key created by ATCMControl for enabling the components is only valid on the PC for which you have requested the registration key.

There are two ways to register your ATCMControl components:

Temporary registration:
Select this registration type to test ATCMControl for a specific time without restrictions of functionality. After a specific period of time, the registration key sent to you by ATCMControl expires and ATCMControl runs in demo mode again.

Permanent registration:
If you have decided to use ATCMControl in your specific area of application in the long term, select this registration type. The registered components can be used without restrictions of functionality and without a time limit.

Click here to go to the description of the registering procedure.

Apart from a machine-dependent registering of ATCMControl components, you can also obtain a hard key for machine-independent registering. The hard key can be connected to the parallel port or USB port (only Windows 98 and Windows 2000). The ATCMControl components which you have installed in that PC and for which the hard key has been registered are then enabled for unlimited use. With the hard key, you can enable ATCMControl installations for unlimited use alternately on different PCs.

To order a hard key, you do not have to create an installation key as described in the ATCMControl Registration section. Indicate the order number and license number in your order and send it to support@4control.de.
