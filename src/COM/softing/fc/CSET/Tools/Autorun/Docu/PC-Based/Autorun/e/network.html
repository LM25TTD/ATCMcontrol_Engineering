Installation of the Network Adapter
Preparations

If you have not yet installed a network on your computer, you need to install one. If there is no network available you must simulate a network by installing a special network adapter, the Loopback Adapter.

Because of the big amount of various network controllers used in computer systems today, we would like to refer to the manufacturers installation guide of your network controller.

The Loopback Adapter can be installed using windows' Hardwareinstallation assistant.
