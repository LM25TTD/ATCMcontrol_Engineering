Configuring the PROFIbus Driver with the
PROFIBUS Control Panel

Use the PROFIBUS control panel applet to configure the parameters of the PROFIBUS hardware device driver for Windows 2000/XP. The PROFIBUS control panel applet is installed in the Control Panel during the installation procedure. To start the PROFIBUS control panel applet, simply double click the "PROFIBUS" icon.

Overview

The user interface of the PROFIBUS control panel behaves like a standard Windows MDI (Multiple Document Interface) application. It consists of the PROFIBUS tree, buttons and an information area displaying hardware-specific configuration parameters.

PROFIBUS Tree Structure

The PROFIBUS tree displays all configured PROFIBUS nodes with their current states, subdivided into the following hardware (board) categories:

  • PROFIboard-ISA (ISA version of PROFIboard with PROFIBUS DP/FMS Master or DP Slave stack)
  • PROFI104 (PC104 board with PROFIBUS DP/FMS Master or DP Slave stack)
  • PROFI104-S (PC104 board with PROFIBUS DP Slave stack)
  • PROFIboard-PCI (PCI version of PROFIboard with PROFIBUS DP/FMS Master or DP Slave stack)
  • PROFIcard / PROFIcard 2 (PC card with PROFIBUS DP/FMS Master stack)
  • FG Series / PROFIgate (PROFIBUS Ethernet Gateway with DP/FMS Master or DP Slave stack)
These categories form the first level of the PROFIBUS tree and indicate the possible types of hardware interfaces. They appear at any time regardless of whether such hardware interfaces are actually installed or not.

A PROFIBUS node is a PROFIBUS device acting as a master or slave. PROFIboard-ISA, PROFIcard / PROFIcard 2, PROFI104 and FG Series / PROFIgate can act as single nodes, PROFIboard-PCI has two channels and can therefore act as two nodes.

When a PROFIBUS node of a certain hardware category has been configured, the category will be displayed in bold letters and a node icon will appear below that category. A configured node is a node known to the PROFIBUS Windows 2000/XP driver. Depending on the state of the node, three different icons are used to display a node:

  • The icon indicates that the PROFIBUS node is working properly.
  • The icon indicates that the PROFIBUS node is not working properly.
  • The icon indicates that the PROFIBUS node is configured but not yet checked.
Information Area

The main settings (firmware type, firmware version, resources, device names, etc.) of the PROFIBUS node configuration are displayed in the information area.

Status Bar

The current status of a PROFIBUS node is displayed in the status bar.

Buttons

The buttons in the PROFIBUS Control Panel have the following meanings:

Add Click the Add button to add a new PROFIBUS node to the PROFIBUS tree.
Edit Click the Edit button to modify the settings of an existing PROFIBUS node in the PROFIBUS tree.
Remove Click the Remove button to remove a PROFIBUS node from the PROFIBUS tree.
Scan Nodes Click the Scan Nodes button to automatically search and configure all connected PROFIBUS interfaces (boards) in the PC.
OK Click the OK button to save the current configuration settings and to load the device drivers.
Cancel Click the Cancel button to restore the last configuration settings and to load the device drivers.
Apply Click the Apply button to apply the configuration settings and load the device drivers.
The procedure for adding or editing a node or for automatically scanning for nodes will be explained in detail in the following sections.

Scanning for Nodes

When you click the Scan Nodes... button, the PROFIBUS hardware device driver searches for all PROFIBUS interfaces (boards) installed in your PC. All necessary resources like dual-port RAM, IO ports, interrupts, etc. will be determined automatically by the hardware device driver.

Depending on the hardware configuration, the scan process may take some time, which is indicated by a clock symbol.

Adding and Editing a PROFIBUS Interface

Click the Add button to configure the hardware device driver for a new PROFIBUS interface.
At first, you will be asked to select the type of the interface board you want to configure.
You can then enter a symbolic name for the new node.
The remaining dialog boxes depend on the type of the interface board. They are explained in detail in the sections below.
Using the Edit button, an existing hardware device driver configuration of a PROFIBUS interface board can be modified. You may modify the node name and other parameters which depend on the type of the interface board.
The dialog boxes for modifying a node are the same as for adding a new node. They are explained in detail in the sections below.

Selecting a Node Name

To access a logical PROFIBUS device from an application, a unique Win32 device name is required. In addition to the standard Win32 device name which is assigned automatically by the PROFIBUS hardware device driver, a Win32 alias device name can be defined optionally.

Enter a symbolic node name in the text field and click the Next button to continue the configuration. Click the Cancel button to abort the configuration and return to the main dialog box.

Selecting the Operating Mode

A PROFIBUS interface can be operated either as DP / FMS master or DP slave device.

Select the desired operating mode and click the Next button to continue the configura-tion. Click the Back button to return to the previous dialog box. Click the Cancel button to abort the configuration and return to the main dialog box.

Note: PROFIcard / PROFIcard 2 supports only the DP / FMS master operating mode.
PROFI104-S supports only the DP slave operating mode.
Caution: Changing the operating mode causes the hardware driver to download the selected firmware to the PROFIBUS board. The existing firmware on the board will be overwritten!

PROFIboard-ISA, PROFI104 and PROFI104-S Parameter Settings

To access PROFIboard-ISA, PROFI104 or PROFI104-S, the PROFIBUS hardware device driver has to allocate resources like I/O ports, address space for the dual-port RAM and interrupt lines.

 

IO-Port Address: The I/O base address of the board. The auto detection feature searches only the range between 0x200 and 0x3FC. To specify an address outside this range, enter the base address in the text field. Refer to the hardware installation manual for more information about valid I/O base addresses.
DP RAM Address: Address of the dual-port memory.
Interrupt: IRQ line.

When you select auto, the PROFIBUS hardware driver determines the necessary resources of the PROFIBUS board.

Select the resources and click the Finish button to complete the configuration and return to the main dialog box. Click the Back button to return to the previous dialog box. Click the Cancel button to abort the configuration and return to the main dialog box.

PROFIcard / PROFIcard 2 Parameter Settings

PROFIcard / PROFIcard 2 Software Interface

The PROFIBUS hardware device driver provides two mechanism for accessing PROFIcard / PROFIcard 2:

  • The standard Windows 2000/XP enabling mechanism for PC Cards
  • PC Card software "CardWare for Windows 2000/XP" by Award Software International Inc.

When using the standard Windows 2000/XP enabling mechanism for PC Cards, only one PROFIcard / PROFIcard 2 can be used. The PROFIcard / PROFIcard 2 must be inserted at boot time.

Select the desired PC Card software interface and click the Next button to continue the configuration. Click the Back button to return to the previous dialog box. Click the Cancel button to abort the configuration and return to the main dialog box.

PROFIcard / PROFIcard 2 (NT Standard)

When accessing PROFIcard / PROFIcard 2 using the Windows 2000/XP PC Card enabler, the PROFIBUS hardware device driver has to allocate resources to PROFIcard / PROFIcard 2.

 

Attribute Memory PC Card attribute memory.
DP RAM Address Address of the dual-port memory.
Interrupt IRQ line.

The PROFIBUS hardware device driver does not feature an auto detection function for the resources, except for the interrupt resource. You have to specify the physical values. You can configure only one "PROFIcard / PROFIcard 2 (NT standard)" because the standard NT PC Card enabling mechanism supports only one PROFIcard / PROFIcard 2. You cannot configure a mixture of both PROFIcard / PROFIcard 2 solutions (PROFIcard / PROFIcard 2 (NT standard) and PROFIcard / PROFIcard 2 (Cardware)) with one installation.

Select the resources and click the Finish button to complete the configuration and return to the main dialog box. Click the Back button to return to the previous dialog box. Click the Cancel button to abort the configuration and return to the main dialog box.

Attention: The Windows 2000/XP PC Card device driver starts during the boot process. Make sure to configure only free resources because your system might not boot with an inserted card (e.g., if you have configured the PROFIcard / PROFIcard 2 (NT standard) using the interrupt of the SCSI controller, the system will hang-up during start-up).

PROFIcard / PROFIcard 2 (Cardware)

When accessing PROFIcard / PROFIcard 2 using the "Award Cardware for Windows 2000/XP", the PROFIBUS hardware device driver needs to allocate resources to PROFIcard / PROFIcard 2.

 

PCMCIA Socket: Socket number of the PROFIcard / PROFIcard 2. This value is only needed to link an interface (board) number of the driver to the PC Card slot number where the PROFIcard / PROFIcard 2 is inserted. When a PROFIcard / PROFIcard 2 is inserted into the PC Card slot, the driver searches the parameters for a PROFIcard / PROFIcard 2 configured for this socket.
DP RAM Address: Address of the dual-port memory.
Interrupt: The PROFIBUS hardware driver automatically determines the IRQ line.

Select the resources and click the Finish button to complete the configuration and return to the main dialog box. Click the Back button to return to the previous dialog box. Click the Cancel button to abort the configuration and return to the main dialog box.

PROFIboard-PCI Parameter Settings

A PROFIboard-PCI interface contains one or two channels, where each channel represents a node in the PROFIBUS network.

 

PCI Slot PCI slot number where the interface is plugged in.
Channel Channel number (only with the dual channel PROFIboard-PCI board).

Click the Finish button to complete the configuration and return to the main dialog box. Click the Back button to return to the previous dialog box. Click the Cancel button to abort the configuration and return to the main dialog box.

FG Series / PROFIgate Parameter Settings

FG Series / PROFIgate Address

The PROFIBUS driver for Windows 2000/XP is capable of accessing a remote FG Series / PROFIgate PROFIBUS Ethernet Gateway over a TCP/IP network and to treat it like a local PROFIBUS interface. To do this, the driver needs to know the IP address or the IP host name of the FG Series / PROFIgate.

Enter the host name (if the IP network has a name server) or the IP address assigned to the FG Series / PROFIgate in the text field and click the Next button to continue the configuration. Click the Cancel button to abort the configuration and return to the main dialog box.

Timeout Parameters for FG Series / PROFIgate

For proper communication with FG Series / PROFIgate, it is necessary to define some timeout values.

 

Timeout for Connect Time control interval for monitoring the connection establishment to FG Series / PROFIgate.
Max Idle Time Time control interval for monitoring the connection.
DP Update Interval Update interval for exchanging the DP process image. The process image will only be transferred if data have changed.
Forced DP Update Interval Forced update interval for exchanging the DP process image (regardless of whether data have changed).

Enter the desired values and click the Finish button to complete the configuration and return to the main dialog box. Click the Back button to return to the previous dialog box. Using the Cancel button you can abort the configuration and return to the main dialog box.

Removing a PROFIBUS Interface

Click the Remove button to remove a PROFIBUS interface from the PROFIBUS tree. After removing an interface click the Apply button to apply the new configuration and to reload the device drivers.
