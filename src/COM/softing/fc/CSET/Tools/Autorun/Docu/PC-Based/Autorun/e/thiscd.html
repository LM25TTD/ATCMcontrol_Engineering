Product Philosophy

This CD contains Softing ATCMControl, which is an offspring of Softing's well proven automation products. Based on the tight and effective integration of SoftControl components, OPC and PROFIBUS products, Ethernet TCP/IP fieldbus couplers/controllers and Softing's long-term control experience, Softing ATCMControl is a pure PC-based system which was designed to program, control, visualize and operate common technical processes.

ATCMControl Feature List

Softing 4CONTROL incorporates the following 4 C´s:

  • Control
    Distributed real-time control using off-the-shelf Windows 2000 / XP
    Designed for incremental on-line changes
    Comprehensive libraries for extended control functionality
  • Console
    Small-footprint visualization using Internet technologies
    Access via standard Internet browser
    Zero-engineering due to automatically generated operator screens
  • Configuration
    IEC 61131-3 compliant programming in ST, FBD, SFC and LD.
    Integrated fieldbus configuration
    Wizard-oriented Microsoft state-of-the-art GUI for less engineering
  • Connectivity
    Support of all major fieldbus I/Os by OPC Client
    Optimum integration of PROFIBUS-DP/PA and Modbus IP
    OPC Server for integration in HMI systems
