Examples for Display Layouts

Examples of Invalid Layouts

In the following, you will find a few examples to show how it cannot work. INVALID is indicated as status and the OK button is not available.

The displays nos. 2 and 13 occur twice in the layout!

In this layout, display no. 4 is L-shaped. This is not permitted! A display must have a rectangular or square shape.

Here, display no. 4 again occurs twice, namely in column 1 and in column 4. This is not permissible!

From this example, you can see that the layout may not have a circular shape.

Example of a Valid Layout

In this example, a valid layout, i.e. a layout which can be created, is shown. You can create layouts in nearly any way you like as long as you observe the rules that each display can only be displayed once and that it must have the shape of a square or a rectangle. Circular shapes have to be avoided!
