Registration of ATCMControl

After installation, all installed ATCMControl components are running in demo mode. This mode involves particular restrictions depending on the components you have installed. For instance, you cannot access the I/O addresses of field devices in demo mode. To use ATCMControl without restrictions, you need to register the installed components. Registering is machine-dependent. This means that the registration key created by ATCMControl for enabling the components is only valid on the PC for which you have requested the registration key. There are two ways to register your ATCMControl components:

Temporary registration:
Select this registration type to test ATCMControl for a specific time without restrictions of functionality. After a specific period of time, the registration key sent to you by ATCMControl expires and ATCMControl runs in demo mode again.

Permanent registration:
If you have decided to use ATCMControl in your specific area of application in the long term, select this registration type. The registered components can be used without restrictions of functionality and without a time limit.

Click here to go to the description of the registering procedure.

Machine-independent registering

Apart from a machine-dependent registering of ATCMControl components, you can also obtain a hard key for machine-independent registering. The hard key can be connected to the parallel port or USB port (only Windows 98 and Windows 2000). The ATCMControl components which you have installed in that PC and for which the hard key has been registered are then enabled for unlimited use. With the hard key, you can enable ATCMControl installations for unlimited use alternately on different PCs. To order a hard key, you do not have to create an installation key as described in the ATCMControl Registration section. Indicate the order number and license number in your order and send it to support@4control.de.

Registering Procedure

  1. Call the Softing License Manager by clicking Programs \Softing AG \Softing License Manager in the Windows start menu.

  2. In the Product Name column, the Softing License Manager lists all ATCMControl components that can be installed. The other columns display information on the license and installation states:



    To register your ATCMControl components, click Create Installationkey in the Softing License Manager.

  3. In the next dialog box, enter the host name of the computer in which the components to be registered have been installed and enter your company name. These data are required for creating the installation key. In the Machinename field, the host name of the computer on which you have started the Softing License Manager is displayed by default; the Company field shows the company name found in the Windows registry database. If you want to register ATCMControl components which are installed in a different computer, you can overwrite these fields. The permitted number of characters is limited to a total of 24 characters for both field.



    Click Continue to confirm your entries.

  4. The entries are displayed again in the next dialog box. Check whether they are correct and then click OK.



    To correct the entries, click Retry. This takes you back to the previous dialog box.

  5. When you have confirmed your entries with Continue, an installation key is created for the computer with the specified host name. Send this key to ATCMControl to have the final registration key generated for you.

  6. To do this, click Copy Installationkey to Clipboard to copy the installation key to the Windows clipboard. You can now paste the installation key from the clipboard into Notepad or into your Email program.

  7. Send the installation key together with the order number and license number to support@4control.de. The order number and license number are essential to be able to send you a final registration key as soon as possible.

  8. You will receive the final registration key by Email as soon as possible. When you have received the final registration key, open the Softing License Manager again and enter the key in the Enter Registration Key(s) field.

  9. Then click Enable. The registered ATCMControl components are now registered and the license state is displayed in the list of ATCMControl components. You can now use the registered ATCMControl components without restrictions.
