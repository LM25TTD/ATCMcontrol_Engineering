Using ATCM Console with Win 95/98

To use a ATCM Console connected with a Windows NT domain in Win 95/98, proceed as follows:

Configuration

You have two possibilities of logging in with an NT domain:

  1. In the first case, the user administration of the NT domain knows your Win 95/98 user ID and your password.
  2. In the second case, the user administration of the NT domain does not know your Win 95/98 user ID and your password.

How to inform the NT domain about your user ID and your password

  1. Go to the Start menu in Win 95/98, select Settings and click on System control.
  2. Select Network.
  3. Click on the Configuration client for MS network tab.
  4. In Properties login check, the check box Login in Win NT domain must be activated!

In both cases, you have to configure the access control and execute the program DCOM Configuration.

Configure access control

  1. Go to the Start menu of Win 95/98, select Settings and click on System control.
  2. Select Network.
  3. In the network dialog, click on the Access control tab.
  4. The Access control to user level radio button must be selected! The name of the server by which the lists of users and groups will be obtained has to be appointed.
  5. Click OK to finish the checking process.

Execute the program DCOM configuration.

  1. Click on the following link to start the program: DCOM Configuration; or go to the Start menu in your Win 95/98 desktop and click on Execute... .
    Enter dcomcnfg.exe. The program is started.
  2. In this program, click on the Default Properties tab.
  3. The Enable Distributed COM on this Computer checkbox must be active.

The Default Authentification Level checkbox must be set up according to one of the following settings, depending on your Win 95/98 user ID:

  • If the user administrator of the NT domain knows you, select Connect in this box.
  • If the user administrator of the NT domain does not know you, the Default Authentification Level box must read None.
    • Click on the Default Security tab and deactivate the Enable Remote Connection check box.
