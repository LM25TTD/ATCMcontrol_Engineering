ATCM Console unter Win 95/98 betreiben

Um eine mit einer Windows NT-Domäne verbundene ATCM Console unter Win 95/98 zu nutzen, gehen Sie wie folgt vor:

Konfiguration

Sie haben zwei Möglichkeiten, sich bei einer NT-Domäne anzumelden:

  1. Im ersten Fall kennt die User Administration der NT-Domäne Ihre Win 95/98- Anmeldung und Ihr Kennwort.
  2. Im zweiten Fall kennt die User Administration der NT-Domäne Ihre Win 95/98-Anmeldung und Ihr Kennwort nicht.

Wie informieren Sie die NT-Domäne über Ihre Anmeldung und Ihr Paßwort?

  1. Gehen Sie in Win 95/98 auf das Start-Menü, wählen Sie Einstellungen und klicken Sie auf Systemsteuerung.
  2. Wählen Sie Netzwerk.
  3. Klicken Sie auf die Registerkarte Konfigurations-Client für MS-Netzwerk.
  4. Unter Eigenschaften Login-Prüfung muß das Kontrollkästchen Login in Win NT-Domäne aktiviert sein!

In beiden Fällen müssen Sie die Zugriffssteuerung konfigurieren und das Programm DCOM Configuration ausführen.

Zugriffssteuerung konfigurieren

  1. Gehen Sie im Start-Menü von Win 95/98 auf Einstellungen und klicken Sie auf Systemsteuerung.
  2. Wählen Sie Netzwerk.
  3. Klicken Sie im Dialog Netzwerk auf die Registerkarte Zugriffssteuerung.
  4. Das Optionsfeld Zugriffssteuerung auf Benutzerebene muß angewählt sein! Zusätzlich muß der Name des Servers angegeben werden, von dem die 'Benutzer- und Gruppenlisten' bezogen werden können.
  5. Klicken Sie auf OK, um die Prüfung abzuschließen.

Das Programm DCOM Configuration ausführen

  1. Klicken Sie den folgenden Link, um das Programm zu starten: DCOM Configuration oder gehen Sie auf Ihrem Win 95/98-Desktop auf das Start-Menü und klicken Sie auf Ausführen... .
    Geben Sie dcomcnfg.exe ein. Das Programm wird gestartet.
  2. Klicken Sie in diesem Programm auf die Registerkarte Default Properties.
  3. Das Kontrollkästchen Enable Distributed COM on this Computer muß aktiviert sein.

Das Kontrollkästchen Default Authentification Level muß abhängig von Ihrer Win 95/98-Anmeldung gemäß einer der folgenden Einstellungen eingerichtet sein:

  • Wenn der User Administrator der NT-Domäne Sie kennt, wählen Sie in diesem Feld Connect.
  • Wenn der User Administrator der NT-Domäne Sie nicht kennt, muß das Feld Default Authentification Level auf None eingestellt werden.
    • Klicken Sie auf die Registerkarte Default Security und deaktivieren Sie das Kontrollkästchen Enable Remote Connection.
