F_ERROR_STRING

Retrieves the error text that corresponds to the error number.

Interface

FUNCTION F_ERROR_STRING:STRING
VAR_INPUT
    ERRCD : DINT;
END_VAR

Return value

Value of the type STRING
the error text (in english).

Parameter

INPUT

ERRCD
Value of the type DINT.
The error number.

Description

Retrieves the error text that corresponds to the error number. The error number of  a file operation is found in the member LSTERRCD of  the
F_SYNC_FILE structure.
