F_GET_DISKFREESPACE

Retrieves the free disk space in kilo bytes for specified path.

Interface

FUNCTION F_GET_DISKFREESPACE:DINT
VAR_INPUT
    PATH : STRING;
END_VAR

VAR_OUTPUT
    FREE_KB : DINT;
END_VAR

Return value

Value of the type DINT, the error code, 0 if no error.

Parameter

INPUT

PATH
Value of the type STRING.
The path for which to get the free disk space.

OUTPUT

FREE_KB
Free disk space in kilo bytes.

Description

Retrieves the free disk space in kilo bytes for specified path.
