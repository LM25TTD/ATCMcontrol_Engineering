F_SYNC_CLOSEFILE

Closes an open file.

Interface

FUNCTION F_SYNC_CLOSEFILE:BOOL
VAR_IN_OUT
    FILE        : F_SYNC_FILE;
END_VAR

Return value

Value of the type BOOL
Signalizes the error status.
Possible values:
TRUE: An error has occurred.
FALSE: No error has occurred.

Parameter

IN_OUT

FILE
Instance of the F_SYNC_FILE structure
File object on which the operation is performed.

Description

FA_SYNC_CLOSEFILE closes the file passed to FILE. FA_SYNC_CLOSEFILE must always be called to terminate an access cycle. The file to be closed is represented by an instance of the F_SYNC_FILE structure.
