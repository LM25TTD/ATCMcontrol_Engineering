F_SYNC_COPYFILE

Copies a file from a source pathname to a destination pathname.

Interface

FUNCTION F_SYNC_COPYFILE:BOOL
VAR_INPUT
    STRSOURCENAME   : STRING;

    STRDESTNAME     : STRING;
END_VAR

Return value

Value of the type DINT, the error code, 0 if no error.

Parameter

INPUT


SRCPATHNAME
Value of the type STRING
Source file pathname.
The file name must be indicated by a fully qualified path (for example, \control\Test.txt). If no path is specified, the final position is undefined.

DESTPATHNAME
Value of the type STRING
Name to which the file is to be renamed.
The file name must be indicated by a fully qualified path (for example, \control\Test.txt). If no path is specified, the final position is undefined.
The compiler does not check whether the syntax of the file name and path information is correct or whether they are plausible.

Description

F_SYNC_COPYFILE copies a file the a source pathnema name specified in the parameter SRCPATHNAME to a destination pathname specified in the parameter DESTPATHNAME. The file to e copied must exist.
