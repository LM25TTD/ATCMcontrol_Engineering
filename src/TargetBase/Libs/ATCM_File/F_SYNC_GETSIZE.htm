F_SYNC_GETSIZE

Delivers the size of a file in bytes.

Interface

FUNCTION F_SYNC_GETSIZE:BOOL
VAR_IN_OUT
    FILE        : F_SYNC_FILE;
END_VAR
VAR_OUTPUT
    SIZE        : DINT;
END_VAR

Return value

Value of the type BOOL
Signalizes the error status.
Possible values:
TRUE: An error has occurred.
FALSE: No error has occurred.

Parameter

IN_OUT

FILE
Instance of the F_SYNC_FILE structure
File object on which the operation is performed.

OUTPUT

SIZE
Value of the type DINT.
Contains the size in bytes of the file passed to FILE on conclusion of the operation.

Description

In the parameter SIZE, F_SYNC_GETSIZE delivers the size of the file passed to FILE. The file does not need to be opened before determining the file size, which means that the execution of F_SYNC_GETSIZE is independent of an access cycle.
