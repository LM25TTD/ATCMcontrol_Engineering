Constants describing task status

Constant Value Description
TSK_STATE_ERROR -1 Task in error state.
TSK_STATE_ONCREATION 0 Task on creation (e.g. at incremental download).
TSK_STATE_STOPPED 1 Task execution stopped.
TSK_STATE_RUNNING 2 Task running.
TSK_STATE_BREAK 3 Task stopped at breakpoint.
TSK_STATE_CONTINUED (not used) 4 Task execution continued.
TSK_STATE_STEP 5 Task execution continued in single step mode.
TSK_STATE_DELETED 6 Task deleted.
TSK_STATE_UNLOADED 7 Task removed from memory.

Description

Return values of function TSK_GetState() included in the library ATCM System2 describing task status.
