EVT_GetException

Gets the task index number and the exception index number of the last raised task exception.

Interface

FUNCTION EVT_GetException : WORD
VAR_OUTPUT
    uTask       : INT;
    ulException : DINT;
END_VAR

Return value

Value of type WORD

OK if successful, else error number. (See ErrNo.cst for details.)

Parameters

OUTPUT

uTask
Value of type INT
Task index number of the task who has most recently raised an exception.
ulException
Value of type DINT
Exception number of most recently raised exception. (See Exception.cst for details.)

Description

Gets the task index number and the exception index number of the last raised task exception. This funciton can be used to implement an exception handler within the IEC project.
