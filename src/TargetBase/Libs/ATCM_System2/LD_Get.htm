LD_Get

Retrieves the system load of a BC/FC.

Interface

FUNCTION LD_Get : WORD
VAR_INPUT
    LoadType : WORD;      (* See SysLoad.cst for details.    *)
    TaskID   : UINT;      (* See SysLoad.cst for details.    *)
END_VAR

VAR_OUTPUT
    str      : STRING;
END_VAR

Return value

Value of type DWORD

OK if successful, else error number. (See ErrNo.cst for details.)

Parameter

INPUT

LoadType
Value of type WORD
Type of the system load.
TaskID
Value of type UINT
Task index number of the current task.

OUTPUT

str
Value of type STRING
Contains information for the given system load and task ID.

Description

LD_Get retrieves informations about the current system load. The infos are stores in the parameter str .

This function is not thread safe. Calling the function from different tasks for the same load type may cause unpredictable results!
