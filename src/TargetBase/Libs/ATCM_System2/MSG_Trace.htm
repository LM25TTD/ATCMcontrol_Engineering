MSG_Trace

Sends a message to the Run Time System  trace device.

Interface

FUNCTION MSG_Trace
VAR_INPUT
    sMsg : STRING;
END_VAR

Return value

none

Parameters

INPUT

sMsg
Value of type STRING
Message to be printed to the Run Time System  trace device.

Description

Sends a message to the Run Time System trace device. (The Run Time System  trace device is dependent from the underlying target AddOn type.)
