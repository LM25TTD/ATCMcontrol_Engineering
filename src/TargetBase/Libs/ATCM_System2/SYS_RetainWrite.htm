SYS_RetainWrite

Writes the IEC retain segment to the file system.

Interface

FUNCTION SYS_RetainWrite : WORD

Return value

Value of type WORD

OK if successful, else error number. (See ErrNo.cst for details.)

Parameter

None

Description

SYS_RetainWrite writes the IEC retain segment to the file system. This function is not supported on all targets and it works only if no real physical retentive memory is available.
