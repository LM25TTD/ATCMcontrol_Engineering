SYS_TraceStat

Traces the actual load statistics to the Engineering.

Interface

FUNCTION_BLOCK SYS_TraceStat

VAR_INPUT
    tCycle : TIME := t#5s;
END_VAR

Parameter

INPUT

tCycle
Value of type TIME
Cycle time for tracing.

Description

SYS_TraceStat traces the actual load statistics to the Engineering.

Warning: This function block is not thread safe and creates a significant amount of processor load itself!
