TSK_ClearStatistic

Clears the statistic information of the corresponding task.

Interface

FUNCTION TSK_ClearStatistic : WORD

VAR_INPUT
    sTask : STRING;
END_VAR

Return value

Value of type WORD

OK if successful, else error number. (See ErrNo.cst for details.)

Parameter

INPUT

sTask
Value of type STRING
Task name.

Description

TSK_ClearStatistic clears the statistic information of the task specified by sTask.
