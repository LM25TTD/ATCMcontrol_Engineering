TSK_GetError

Gets the task error state of the corresponding task.

Interface

FUNCTION TSK_GetError : WORD
VAR_INPUT
    sTask   : STRING;
END_VAR
  
VAR_OUTPUT
    ulError : DINT;
END_VAR

Return value

Value of type WORD

OK if successful, else error number. (See ErrNo.cst for details.)

Parameters

INPUT

sTask
Value of type STRING
Corresponding task name.

OUTPUT

ulError
Value of type DINT
Current error state of the named task. (See System2.cst for possible values.)

Description

Gets the task error number of the corresponding task
