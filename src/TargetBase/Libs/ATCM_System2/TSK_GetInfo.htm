TSK_GetInfo

Gets the task information structure of the corresponding task.

Interface

VAR_INPUT
    sTask   : STRING;
END_VAR
VAR_OUTPUT
    TaskInfo : TSK_INFO;
END_VAR

Return value

Value of type WORD

OK if successful, else error number. (See ErrNo.cst for details.)

Parameters

INPUT

sTask
Value of type STRING

OUTPUT

TaskInfo
Value of type TSK_INFO
Task information structure for the named task. (See TSK_INFO.sdt for details.)
