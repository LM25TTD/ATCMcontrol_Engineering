TSK_GetName

Gets the task name of the corresponding task.

Interface

FUNCTION TSK_GetName : WORD
VAR_INPUT
    uTask   : INT;
END_VAR
  
VAR_OUTPUT
    sTask   : STRING;
END_VAR

Return value

Value of type WORD

OK if successful, else error number. (See ErrNo.cst for details.)

Parameters

INPUT

uTask
Value of type INT
Task index number.

OUTPUT

sTask
Value of type STRING
To task index number corresponding string name.

Description

Gets the task name of a task identified by its task index number
