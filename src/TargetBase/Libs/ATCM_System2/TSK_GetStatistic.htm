TSK_GetStatistic

Retrieves statistic information of the corresponding task.

Interface

FUNCTION TSK_GetStatistic : WORD
VAR_INPUT
    sTask       : STRING;
END_VAR

VAR_OUTPUT
    TaskStat    : TSK_STAT;
END_VAR

Return value

Value of type WORD

OK if successful, else error number. (See ErrNo.cst for details.)

Parameter

INPUT

sTask
Value of type STRING
Task name.

OUTPUT

TaskStat
Value of type TSK_Stat
Structure instance of the structure TSK_Stat.

Description

TSK_GetStatistic retrieves statistic information of the task specified by sTask. The information is stored in the parameter TaskStat.
