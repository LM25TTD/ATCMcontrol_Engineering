TSK_INFO

Structure representing a task state.

Interface

TSK_INFO : STRUCT
     ulState     : DINT; (*[Task state]*)
     ulPriority  : DINT; (*[Task Priority]*)
     tCycle      : TIME; (*[Task cycle time (in ms)]*)
     tETMax      : TIME; (*[Task max. execution time (in ms)]*)
     ulError     : DINT; (*[Task error state]*)
END_STRUCT;

Parameters

ulState

Value of type DINT.

Task state. (See task state values in System2.cst.)

ulPriority

Value of type DINT.

Task priorities. This values are the real operating system priorities and not the abstract values specified in the IEC project. (So the possible values can be different on different underlying target types.)

tCycle

Value of type TIME.

Task cycle time for cyclic tasks.

tETMax

Value of type TIME.

Maximum task execution time.

ulError

Value of type DINT.

Task error state (error number). (See task state values in ErrNo.cst.)

Description

Starts the named task. The corresponding data for each structure can be retrieved by calling the TSK_GetInfo() function for a specific task.
