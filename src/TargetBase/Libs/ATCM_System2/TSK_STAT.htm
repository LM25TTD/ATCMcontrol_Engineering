TSK_STAT

Structure representing a task state.

Interface

TYPE
  TSK_STAT : STRUCT
    TaskState      : UDINT;
    CycleTime      : UDINT;
    ExecMax        : UDINT;
    ExecMin        : UDINT;
    ExecAverage    : UDINT;
    ExecCount      : UDINT;
    WDogCounter    : UDINT;
    WDogTrigger    : UDINT;
  END_STRUCT;
END_TYPE

Parameters

TaskState

Value of type UDINT.

Task state. (See task state values in System2.cst.)

CycleTime

Value of type UDINT.

Task cycle time for cyclic tasks.

ExecMax

Value of type UDINT.

Maximum task execution time in us.

ExecMin

Value of type UDINT.

Minimum task execution time in us.

ExecAverage

Value of type UDINT.

Average task execution time in us.

ExecCount

Value of type UDINT.

Task execution counter.

WDogCounter

Value of type UDINT.

Watchdog counter.

WDogTrigger

Value of type UDINT.

Watchdog trigger level.

Description

Provides information about task state, execution and cycle timing and about the watchdog.
The statistic informations can be retrieved for a specific task by calling the function TSK_GetStatistic.
They can be reset by calling the function TSK_ClearStatistic.
