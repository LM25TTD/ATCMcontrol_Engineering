WD_GetTrigger

Retrieves information about the watchdog for the corresponding task.

Interface

FUNCTION WD_GetTrigger : WORD
VAR_INPUT
    sTask        : STRING;
END_VAR

VAR_OUTPUT
    WDTrigger    : UDINT;
    WDCount      : UDINT;
END_VAR

Return value

Value of type WORD

 OK if successful, else error number. (See ErrNo.cst for details.)

Parameters

INPUT

sTask
Value of type STRING
Corresponding task name.

OUTPUT

WDTrigger
Value of type UDINT
Watchdog trigger level.
WDCount
Value of type UDINT
Actual watchdog trigger count.

Description

Gets the watchdog trigger level and the actrual watchdog trigger count of the corresponding task
