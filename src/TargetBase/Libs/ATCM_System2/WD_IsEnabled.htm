WD_IsEnabled

Retrieves the watchdog state (enabled / disabled) for the corresponding task.

Interface

FUNCTION WD_IsEnabled : WORD
VAR_INPUT
    sTask       : STRING;
END_VAR

VAR_OUTPUT
    bEnabled    : BOOL;
END_VAR

Return value

Value of type WORD

 OK if successful, else error number. (See ErrNo.cst for details.)

Parameters

INPUT

sTask
Value of type STRING
Corresponding task name.

OUTPUT

bEnabled
Value of type BOOL
Watchdog enabled if TRUE.

Description

Gets the watchdog state (enabled / disabled) for a given task.
