Datalog_get

Capture data to create single log file and the list of captured labels.

Interface

FUNCTION Datalog_get : UINT
VAR_INPUT
        label : STRING;
        value : REAL;
END_VAR

Return value

Value of type UINT

0 : Succes
1 : log failed
2 : buffer exceeded

Parameter

INPUT

label
Value of type STRING
label of the captured data (20 char max without spaces)
value
Value of type REAL
value of the captured data

Description

Datalog_get Capture data to create single log file and the list of captured labels.
