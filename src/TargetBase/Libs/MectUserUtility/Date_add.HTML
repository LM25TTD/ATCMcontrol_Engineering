Date_add

Perform add operation between two date

Interface

FUNCTION Date_add : STRING
VAR_INPUT
	initDate  : STRING; (*[ starting date in format "DD-MM-YYYY" ]*)
	format    : STRING; (*[ "D" for days, "W" for weeks, "M" for months and "Y" for years ]*)
	increment : INT; (*[ value to add ]*)
END_VAR
VAR_OUTPUT
	finalDate     : STRING; (*[ final date in format "DD-MM-YYYY" ]*)
END_VAR

Return value

Value of type UINT

0 : Success
1 : Error

Parameter

INPUT

initDate
Value of type STRING
starting date in format "DD-MM-YYYY"
format
Value of type STRING
describe in which format is the increment value: "D" for days, "W" for weeks, "M" for months and "Y" for years
increment
Value of type INT
value to add described by format parameter

OUTPUT

finalDate
Value of type STRING
final date in format "DD-MM-YYYY"

Description

Date_add perform add operation between two date
