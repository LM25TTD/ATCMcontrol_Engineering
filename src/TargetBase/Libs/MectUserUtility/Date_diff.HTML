Date_diff

Perform diff operation between two date

Interface

FUNCTION Date_diff : UINT
VAR_INPUT
	initDate  : STRING; (*[ starting date in format "DD-MM-YYYY" ]*)
	finalDate : STRING; (*[ end date in format "DD-MM-YYYY" ]*)
	format    : STRING; (*[ "D" for days, "W" for weeks, "M" for months and "Y" for years ]*)
END_VAR
VAR_OUTPUT
	diff      : DINT; (*[ difference between start date and end date expressed in "format" ]*)
END_VAR

Return value

Value of type UINT

0 : Success
1 : Error

Parameter

INPUT

initDate
Value of type STRING
starting date in format "DD-MM-YYYY"
format
Value of type STRING
describe in which format is the diff value: "D" for days, "W" for weeks, "M" for months and "Y" for years
diff
Value of type INT
value to subtract described by format parameter

OUTPUT

finalDate
Value of type STRING
final date in format "DD-MM-YYYY"

Description

Date_diff perform diff operation between two date
