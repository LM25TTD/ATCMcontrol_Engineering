Function Name

modbus_close - close a Modbus connection

Interface

FUNCTION MODBUS_CLOSE : UINT

VAR_INPUT

ctx : STRING;

END_VAR

Return value

Value of type UINT

0 of success 1 for ERROR

Parameters

INPUT

ctx

Value of type STRING

Context for the connection to be closed. Valid values are RTU, TCP, TCPRTU

Description

The MODBUS_CLOSE function shall close the connection established with the backend set in the context.
