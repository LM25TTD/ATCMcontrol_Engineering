Function Name

modbus_flush - flush non-transmitted data

Interface

FUNCTION MODBUS_FLUSH : UINT

 

VAR_INPUT

ctx: STRING;

END_VAR

Return value

Value of type UINT

0 for success 1 for error

Parameters

INPUT

ctx

Value of type STRING

Context for the connection to be flushed. Valid values are RTU, TCP, TCPRTU

Description

The MODBUS_FLUSH function shall discard data received but not read to the socket or file descriptor associated to the context 'ctx'.
