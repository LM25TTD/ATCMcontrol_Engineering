Function Name

modbus_get_float_CDAB - get a float value from 2 registers

Interface

FUNCTION MODBUS_GET_FLOAT_CDAB : UINT

 

VAR_INPUT

src : ARRAY[0..2] of UINT;

END_VAR

VAR_OUTPUT

value : REAL;

END_VAR

Return value

Value of type UINT

0 for success 1 for error

Parameters

INPUT

src

Value of type ARRAY[0..2] of UINT

Source bytes array

OUTPUT

value

Value of type REAL

Retrieven float value

Description

The MODBUS_GET_FLOAT_CDAB function shall get a float from 4 bytes in inversed Modbus format (CDAB order). The 'src' array must be pointer on two 16 bits values.
