Function Name

modbus_read_input_registers - read many input registers

Interface

FUNCTION MODBUS_READ_INPUT_REGISTERS : UINT

 

VAR_INPUT

ctx:STRING;

addr:UINT;

nb_regs:UINT;

END_VAR

VAR_OUTPUT

data : ARRAY[0..259] of UINT;

END_VAR

Return value

Value of type UINT

0 for success 1 for error

Parameters

INPUT

ctx

Value of type STRING

Context for the current connection. Valid values are RTU, TCP, TCPRTU

addr

Value of type UINT

Remote device address to be read

nb_regs

Value of type UINT

Number of registers to be read

OUTPUT

data

Value of type ARRAY[0..259] of UINT

Array where retrieven registers are stored

Description

The MODBUS_READ_INPUT_REGISTERS function shall read the content of the 'nb_regs' input registers to address 'addr' of the remote device. The result of the reading is stored in 'data' array as word values (16 bits). 'data' must be at least 'nb_regs' * sizeof(uint16_t) to store the results The function uses the Modbus function code 0x04 (read input registers).
