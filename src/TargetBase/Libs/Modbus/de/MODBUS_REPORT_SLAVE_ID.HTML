Function Name

modbus_report_slave_id - returns a description of the controller

Interface

FUNCTION MODBUS_REPORT_SLAVE_ID : UINT

 

VAR_INPUT

ctx:STRING;

END_VAR

VAR_OUTPUT

data : ARRAY[0..259] of BYTE;

END_VAR

Return value

Value of type UINT

0 for success 1 for error

Parameters

INPUT

ctx

Value of type STRING

Context for the current connection. Valid values are RTU, TCP, TCPRTU

OUTPUT

data

Value of type ARRAY[0..259] of BYTE

Array where retrieven data is stored

Description

The MODBUS_REPORT_SLAVE_ID function shall send a request to the controller to obtain a description of the controller. The response stored in 'data' contains:

* the byte count of the response

* the slave ID, this unique ID is in reality not unique at all so it's not possible to depend on it to know how the information are packed in the response.

* the run indicator status (0x00 = OFF, 0xFF = ON)

* additional data specific to each controller. For example, libmodbus returns the version of the library as a string.
