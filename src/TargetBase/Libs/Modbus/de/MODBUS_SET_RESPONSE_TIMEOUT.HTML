Function Name

modbus_set_response_timeout - set timeout for response

Interface

FUNCTION MODBUS_SET_RESPONSE_TIMEOUT : UINT

 

VAR_INPUT

ctx:STRING;

timeout_ms:UINT;

END_VAR

Return value

Value of type UINT

0 for success 1 for error

Parameters

INPUT

ctx

Value of type STRING

Context for the current connection. Valid values are RTU, TCP, TCPRTU

timeout_ms

Value of type UINT

timeout expressed in ms to be set

Description

The MODBUS_SET_RESPONSE_TIMEOUT function shall set the timeout interval used to wait for a response. If the waiting before receiving the response is longer than the given timeout, an error will be raised.
