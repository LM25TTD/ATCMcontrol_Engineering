Function Name

modbus_write_bit - write a single bit

Interface

FUNCTION MODBUS_WRITE_BIT : UINT

 

VAR_INPUT

ctx:STRING;

addr:UINT;

status: INT;

END_VAR

Return value

Value of type UINT

0 for success 1 for error

Parameters

INPUT

ctx

Value of type STRING

Context for the current connection. Valid values are RTU, TCP, TCPRTU

addr

Value of type DINT

Remote device starting address to be written

status

Value of type INT

Content to be written

Description

The MODBUS_WRITE_BIT function shall write the status of 'status' at the address 'addr' of the remote device. The value must be set to TRUE or FALSE. The function uses the Modbus function code 0x05 (force single coil).
