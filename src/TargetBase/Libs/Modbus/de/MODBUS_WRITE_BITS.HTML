Function Name

modbus_write_bits - write many bits

Interface

FUNCTION MODBUS_WRITE_BITS : UINT

 

VAR_INPUT

ctx:STRING;

addr:UINT;

nb_bits: UINT;

data: ARRAY[0..259] of BYTE;

END_VAR

Return value

Value of type UINT

0 for success 1 for error

Parameters

INPUT

ctx

Value of type STRING

Context for the current connection. Valid values are RTU, TCP, TCPRTU

addr

Value of type UINT

Remote device starting address to be written

nb_bits

Value of type UINT

number of bits to be written

data

Value of type ARRAY[0..259] of BYTE

Content to be written

Description

Tfunction shall write the status of the 'nb_bits' bits (coils) from 'data' at the address 'addr' of the remote device. The 'data' array must contains bytes set to TRUE or FALSE. The function uses the Modbus function code 0x0F (force multiple coils).
