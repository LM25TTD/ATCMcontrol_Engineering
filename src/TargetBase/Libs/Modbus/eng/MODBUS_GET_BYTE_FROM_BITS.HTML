Function Name

modbus_get_byte_from_bits - get the value from many bits

Interface

FUNCTION MODBUS_GET_BYTE_FROM_BITS : UINT

 

VAR_INPUT

src: ARRAY[0..259] of BYTE;

index: UINT;

nb_bits: UINT;

END_VAR

VAR_OUTPUT

dest:BYTE;

END_VAR

Return value

Value of type UINT

0 for success 1 for error

Parameters

INPUT

src

Value of type ARRAY[0..259] of BYTE

Source bit array from which retrieve data

index

Value of type UINT

Position in the array from which starting to get bit

nb_bits

Value of type UINT

Number of bit to retrieve

OUTPUT

dest

Value of type BYTE

Byte to be filled with retrieven bit

Description

The MODBUS_GET_BYTE_FROM_BITS function shall extract a value from many bits. All 'nb_bits' bits from 'src' at position 'index' will be read as a single value. To obtain a full byte, set nb_bits to 8.
