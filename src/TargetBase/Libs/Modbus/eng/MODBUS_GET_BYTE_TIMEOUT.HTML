Function Name

modbus_get_byte_timeout - get timeout between bytes

Interface

FUNCTION MODBUS_GET_BYTE_TIMEOUT : UINT

 

VAR_INPUT

ctx:STRING;

END_VAR

VAR_OUTPUT

timeout_ms:UINT;

END_VAR

Return value

Value of type UINT

0 for success 1 for error

Parameters

INPUT

ctx

Value of type STRING

Context for the current connection. Valid values are RTU, TCP, TCPRTU

OUTPUT

timeout_ms

Value of type UINT

Retrieven timeout between bytes expressed in ms

Description

The MODBUS_GET_BYTE_TIMEOUT function shall store the timeout interval between two consecutive bytes of the same message in the 'timeout_ms' argument.
