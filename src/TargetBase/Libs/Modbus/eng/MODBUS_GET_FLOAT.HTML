Function Name

modbus_get_float - get a float value from 2 registers

Interface

FUNCTION MODBUS_GET_FLOAT : UINT

 

VAR_INPUT

src : ARRAY[0..2] of UINT;

END_VAR

VAR_OUTPUT

value : REAL;

END_VAR

Return value

Value of type UINT

0 for success 1 for error

Parameters

INPUT

src

Value of type ARRAY[0..2] of UINT

Source bytes array

OUTPUT

value

Value of type REAL

Retrieven float value

Description

The MODBUS_GET_FLOAT function shall get a float from 4 bytes in Modbus format (ABCD order). The 'src' array must be pointer on two 16 bits values, for example, if the first word is set to 0x4465 and the second to 0x229a, the float value will be 916.540649.
