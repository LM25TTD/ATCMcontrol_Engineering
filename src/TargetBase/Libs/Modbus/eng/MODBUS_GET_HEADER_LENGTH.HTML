Function Name

modbus_get_header_length - retrieve the current header length

Interface

FUNCTION MODBUS_GET_HEADER_LENGTH : UINT

 

VAR_INPUT

ctx:STRING;

END_VAR

Return value

Value of type UINT

0 for success 1 for error

Parameters

INPUT

ctx

Value of type STRING

Context for the current connection. Valid values are RTU, TCP, TCPRTU

Description

The GET_HEADER_LENGTH function shall retrieve the current header length from the backend. This function is convenient to manipulate a message and so its limited to low-level operations.
