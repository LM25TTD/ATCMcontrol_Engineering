Function Name

modbus_read_input_bits - read many input bits

Interface

FUNCTION MODBUS_READ_INPUT_BITS : UINT

 

VAR_INPUT

ctx:STRING;

addr:UINT;

nb_bits:UINT;

END_VAR

VAR_OUTPUT

data : ARRAY[0..259] of BOOL;

END_VAR

Return value

Value of type UINT

0 for success 1 for error

Parameters

INPUT

ctx

Value of type STRING

Context for the current connection. Valid values are RTU, TCP, TCPRTU

addr

Value of type UINT

Remote device address to be read

nb_bits

Value of type UINT

Number of bits to be read

OUTPUT

data

Value of type ARRAY[0..259] of BOOL

Array where retrieven bits are stored

Description

The MODBUS_READ_INPUT_BITS function shall read the status of the 'nb_bits' input bits to the address 'addr' of the remote device. The result of reading is stored in 'data' array as unsigned bytes (8 bits) set to TRUE or FALSE. 'data' must be at least 'nb' * sizeof(uint8_t) to store the results. The function uses the Modbus function code 0x02 (read input status).
