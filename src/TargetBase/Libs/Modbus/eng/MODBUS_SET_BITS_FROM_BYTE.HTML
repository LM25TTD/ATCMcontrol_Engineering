Function Name

modbus_set_bits_from_byte - set many bits from a single byte value

Interface

FUNCTION MODBUS_SET_BITS_FROM_BYTE : UINT

 

VAR_INPUT

index: UINT;

value: BYTE;

END_VAR

VAR_OUTPUT

dest:ARRAY[0..259] of BYTE;

END_VAR

Return value

Value of type UINT

0 for success 1 for error

Parameters

INPUT

index

Value of type UINT

Position in the array from which starting to set bits

value

Value of type UINT

source byte

OUTPUT

dest

Value of type ARRAY[0..259] of BYTE

Bits array to be filled

Description

The MODBUS_SET_BITS_FROM_BYTE function shall set many bits from a single byte. All 8 bits from the byte 'value' will be written to 'dest' array starting at 'index' position.
