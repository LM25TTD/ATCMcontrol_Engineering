Function Name

modbus_set_float_dcba - set a float value in 2 registers using DCBA byte order

Interface

FUNCTION MODBUS_SET_FLOAT_DCBA : UINT

 

VAR_INPUT

value : REAL;

END_VAR

VAR_OUTPUT

dest : ARRAY[0..2] of UINT;

END_VAR

Return value

Value of type UINT

0 for success 1 for error

Parameters

INPUT

value

Value of type REAL

float value to be set

OUTPUT

dest

Value of type ARRAY[0..2] of UINT

Destination bytes array

Description

The MODBUS_SET_FLOAT_DCBA function shall set a float to 4 bytes in inversed Modbus format (DCBA order). The 'dest' array must be pointer on two 16 bits values to be able to store the full result of the conversion.
