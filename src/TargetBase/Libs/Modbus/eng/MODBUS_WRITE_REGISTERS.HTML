Function Name

modbus_write_registers - write many registers

Interface

FUNCTION MODBUS_WRITE_REGISTERS : UINT

 

VAR_INPUT

ctx:STRING;

addr:UINT;

nb_reg: UINT;

data: ARRAY[0..259] of UINT;

END_VAR

Return value

Value of type UINT

0 for success 1 for error

Parameters

INPUT

ctx

Value of type STRING

Context for the current connection. Valid values are RTU, TCP, TCPRTU

addr

Value of type UINT

Remote device starting address to be written

nb_reg

Value of type UINT

Number of registers to be written

data

Value of type ARRAY[0..259] of UINT

Content to be written

Description

The MODBUS_WRITE_REGISTERS function shall write the content of the 'nb_reg' holding registers from the array 'data' at address 'addr' of the remote device. The function uses the Modbus function code 0x10 (preset multiple registers).
