SDOGetStatus

Get the CAN open status

Interface

FUNCTION SDOGetStatus : UINT
VAR_OUTPUT
	status : UINT; (*[status 0: not ready; <> 0 : ready]*)
END_VAR

Return value

Value of type UINT

0 : Success
1 : Error

Parameter

OUTPUT

status
Value of type UINT
CAN open status: 0 if is not ready not ready; <> 0 if it is ready

Description

SDOGetStatus get the CAN open status
