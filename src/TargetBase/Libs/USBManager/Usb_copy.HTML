Usb_copy

Copy either a file or a directory with all its content between usb keys.

Interface

FUNCTION Usb_copy : UINT
VAR_INPUT
	srcfile : STRING;
	dstfile : STRING;
	src_index : UINT;
	dst_index : UINT;
END_VAR

Return value

Value of type UINT

0 : Success
1 : cannot mount/umount usb key
2 : source file does not exist
3 : there's no enough space on destination usbkey to perform the copy
4 : copy failed

Parameter

INPUT

srcfile
Value of type STRING
Name of the source file
dstfile
Value of type STRING
Name of the destination file
src_index
Value of type UINT
USB key access index or the source file
dst_index
Value of type UINT
USB key access index or the destination file

Description

Usb_copy Copy either a file or a directory with all its content between usb keys. The status of execution handling is stored in USBfeedback[1]
