Usb_delete

Delete a file or a directory from a usb key.

Interface

FUNCTION Usb_delete : UINT
VAR_INPUT
	file : STRING;
	index : UINT;
END_VAR

Return value

Value of type UINT

0 : Success
1 : Error

Parameter

INPUT

file
Value of type STRING
file o directory to be deleted
index
Value of type UINT
index representing the usbkey on which the file is stored

Description

Usb_delete Delete a file or a directory from a usb key
