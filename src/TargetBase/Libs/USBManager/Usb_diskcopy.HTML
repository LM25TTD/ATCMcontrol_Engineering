Usb_diskcopy

Copy either a file or a directory with all its content from a usb key to internal flash

Interface

FUNCTION Usb_diskcopy : UINT
VAR_INPUT
	srcfile : STRING;
	dstfile : STRING;
	index : UINT;
END_VAR

Return value

Value of type UINT

0 : Success
1 : resource already busy cannot start another usb operation

Parameter

INPUT

srcfile
Value of type STRING
Name of file o directory to be copied
dstfile
Value of type STRING
Name of file o directory for destination
index
Value of type UINT
index in the USBstatus array representing the source usbkey for filename

Description

Usb_diskcopy Copy either a file or a directory with all its content from a usb key to internal flash
