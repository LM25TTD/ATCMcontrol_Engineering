Usb_mkdir

Create a directory on a usb key

Interface

FUNCTION Usb_mkdir : UINT
VAR_INPUT
	dirname : STRING;
	index : UINT;
END_VAR

Return value

Value of type UINT

0 : Success
1 : error
2 : directory already exist

Parameter

INPUT

dirname
Value of type STRING
Directory to be created
index
Value of type UINT
index representing the usbkey on which the directory will exist

Description

Usb_mkdir Create a directory on a usb key
